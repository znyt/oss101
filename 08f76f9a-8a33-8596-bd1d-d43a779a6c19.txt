　　嗖…
<br/>　　萧逸身影一动，速度快到了极点。
<br/>　　九万三千道星光之力，早已加身。
<br/>　　“好快。”在场长老，除了少数几人外，其余人等，尽皆脸色一变。
<br/>　　“小贼，尔敢？”
<br/>　　“好猖狂的小子，便让老夫来会一会你。”
<br/>　　刚才说话的长老，一掌轰出。
<br/>　　“滚。”萧逸暴喝一声。
<br/>　　手中冷焱剑，星光涌动下，威势滔天。
<br/>　　率先攻来的长老，瞬间掌风溃散，被重重劈飞。
<br/>　　这里的长老，不下三十人。
<br/>　　既然在血光洞府内的天骄，乃是八府天骄。
<br/>　　那么很显然，这三十位长老，也是八府的长老。
<br/>　　不过，这里面大部分都是圣皇境后期武者罢了，有一些达到圣皇境巅峰，少数一些则与天冥府那位大长老气息差不多。
<br/>　　也就是说，这里最强的几人，也不过是掌控九千三百条完整武道的层次罢了。
<br/>　　萧逸还不惧之。
<br/>　　锵…
<br/>　　散发冷异白光的冷焱剑，直取其中一位长老而去。
<br/>　　这位长老，便是其中最强的几人之一。
<br/>　　“找死。”老者手中一把利剑祭出。
<br/>　　很显然，这是个剑修，乃是剑光府强者。
<br/>　　锵…
<br/>　　又是一声锵鸣，不过这一声，却比之前的一声要清脆得多。
<br/>　　这是两把利剑的交锋之音。
<br/>　　两人的交锋，竟是平手。
<br/>　　不过，这平手之势，仅仅持续了数秒。
<br/>　　嘭…一声爆响，这位剑光府武者，被萧逸一剑震开。
<br/>　　“呵，留力了？”萧逸冷笑一声。
<br/>　　嗖…嗖…嗖…
<br/>　　萧逸的身影，化作了一阵幻影。
<br/>　　手中之剑，霎时势如破竹，无人可挡。
<br/>　　嘭…嘭…嘭…
<br/>　　一声声爆响之下，不过片刻，三十位长老，悉数落败。
<br/>　　“倒是有些本事。”一众长老，脸色难看无比。
<br/>　　他们确实是留力了，并未实力尽出，否则萧逸再强也不可能这么快将他们悉数击败。
<br/>　　不过萧逸很清楚，他们留力，并非是对他手下留情。
<br/>　　而是，这些长老，需要保持全盛状态，留有充足的元力，之后为自家天骄引导。
<br/>　　如今，八府天骄不过是在洞府内凝聚血脉之力。
<br/>　　待得血脉之力发挥到极致，武道圣丹悉数消化，这些长老便会进入洞府，替自家天骄引导力量。
<br/>　　“看来你们是拦不住我了。”萧逸不屑一笑。
<br/>　　“走吧。”萧逸看向顾怜星二人。
<br/>　　顾怜星二人，快步走向萧逸。
<br/>　　不过，二人来到萧逸身旁后，亦忽然止步，未再动弹。
<br/>　　“怎么？”萧逸皱了皱眉。
<br/>　　顾怜星脸色迟疑，半晌，道，“怜星只是觉得，萧逸公子硬闯洞府，此举，是否有些过分了？”
<br/>　　顾怜星语气中带着迟疑。
<br/>　　但其中，并无半分责怪，只是低声与萧逸交谈着。
<br/>　　“过分？”萧逸冷笑一声，“我明白怜星姑娘的意思。”
<br/>　　“但我只问，八府武者，凭什么不让我进洞府？”
<br/>　　“十八府险地，虽是以十八府命名，但这便能代表这些险地都是十八府的囊中之物？”
<br/>　　“十八府各府坐落地域，距离各险地，多远？”
<br/>　　“拿这血光险地来说，乃是地域边缘；而血光府本身，坐落于周遭十数地域的中间。”
<br/>　　“二者起码隔了数个地域距离，何止数百万里之遥？”
<br/>　　“便是因为血光险地附近地域，最强势力乃是血光府，这偌大片妖兽森林，便归血光府了？”
<br/>　　“这无边无际的妖兽森林，便成了他们血光府的后花园了？”
<br/>　　“是不是我萧逸占了这些险地，实力比他们血光府强横，这血光险地，便归我萧逸了？”
<br/>　　萧逸冰冷的话语，一句句道出。
<br/>　　周遭长老，脸色愤怒，“小贼，你这是强词夺理。”
<br/>　　萧逸身旁，公孙火舞亦迟疑道，“那个…萧逸，我也觉得你有些强词夺理了。”
<br/>　　公孙火舞见萧逸脸色冰冷，并未如往常那般，称呼萧逸‘小贼’，而是以名字称呼之，语气呐呐。
<br/>　　她知道，此刻的萧逸，显然是动了怒。
<br/>　　萧逸的目光，看向公孙火舞，“我知道你想说什么。”
<br/>　　“传言，十八府的创立者，曾在这些险地修炼，之后才有了这些各有异状的险地。”
<br/>　　“但传言归传言，先不说是真是假，且说这些洞府，便能因此而认可为是十八府的创立者所留？”
<br/>　　“呵，无数岁月了，我可没见十八府的武者能掌控这些洞府。”
<br/>　　拿天星洞府来说，那是天星府历史上最强的一位府主，拼了性命才取出了一盏天星盏。
<br/>　　天星盏，也因此才有了些许掌控洞府禁制的能力。
<br/>　　若真说这些洞府是十八府的老祖宗所留，那岂非这些创立者故意要取自家子孙后辈性命？
<br/>　　若这些洞府，真是十八府所有；在场的血光府长老，大可调动血光府内禁制，轰他萧逸出血光险地。
<br/>　　“额，这个…”公孙火舞与顾怜星霎时哑口无言。
<br/>　　“哼。”远处，一众长老愈加愤怒。
<br/>　　“小贼，你倒是巧如舌簧…”
<br/>　　萧逸冷冷打断道，“你们平日里霸道，与我无关。”
<br/>　　“但天大地大，无数险地，无数妖兽森林，暂时还未有一个能拦我萧逸的脚步。”
<br/>　　冷傲的话语，响彻周遭。
<br/>　　在场长老，脸色愤愤，却又面面相觑。
<br/>　　萧逸并未理会，带着顾怜星二人直接入了洞府。
<br/>　　进入血光洞府。
<br/>　　果然，率先映入眼帘的，便是一巨大广场。
<br/>　　里头景象，与天星洞府和幻光洞府之内，一模一样。
<br/>　　广场之内，八位天骄，盘膝而坐，闭目修炼。
<br/>　　一道道血色，萦绕在他们周遭。
<br/>　　很显然，他们在准备爆发血脉之力。
<br/>　　而这些萦绕的血色，其尽头，正是广场尽头，主府之内。
<br/>　　主府之内，一个浑圆的血色‘圆球’，刺眼至极。
<br/>　　圆球之上，一股股精纯浑厚的气血力量，让人闻之色变，心惊胆寒。
<br/>　　萧逸看了眼广场中央的八人，淡淡道，“不打扰他们八人便是，我们直接去主府。”
<br/>　　八人的周遭，事实上早有阵法禁制包裹；等闲的噪音、骚乱等等，根本影响不了八人。
<br/>　　八人作为八府的少府主，又是准备着第二次的武魂觉醒这等大事，八府的强者怎可能不提前布置好阵法以作保护？
<br/>　　除非萧逸三人根本是来故意捣乱的，强行破了他们八人周遭的阵法。
<br/>　　否则，他们三人进入，根本不可能扰乱这八人的突破。
