　　第292章
<br/>　　蛤蟆寨。
<br/>　　诸人聚在一起，张张脸上写满不可思议。彼此目光对视，皆不知该如何开口。
<br/>　　“这个……不会是开玩笑吧。”徐神机开口，心中觉得自己想明白了：“一定是开玩笑了，诓我们说自己被劫走了，待会儿大当家就自己冒了出来，大当家还真是童心未泯呐。”
<br/>　　“来来，等大当家冒出来时，大家配合一下，不要让他太失望。”刘悲。
<br/>　　高飞豹看着诸人，一副『快别扯淡』的表情。
<br/>　　“大当家真的被劫走了，你们怎么就不信呢，对方还留了信。”高飞豹。
<br/>　　落玉寨，樊梨花。
<br/>　　刚才大家都已看过这张纸条，此时就在桌上搁着。
<br/>　　“该不会吧，大当家就是山贼，还能被山贼劫走，多丢人啊。”徐神机。
<br/>　　“大当家也是这么说的啊，可他拉泡屎的功夫就不见了。”
<br/>　　“这么说……大当家拉屎的时候把自己拉没了。”刘悲一脸匪夷所思，又拿起那张纸条看看。
<br/>　　“喂，他被人劫走了，你们还干坐在这里做什么，赶快想办法救他回来啊。”李婉儿忽地跑进来，焦急道。
<br/>　　“是的，是的，一定要救。”刘悲开口：“关键是……这落玉寨究竟在哪里？”
<br/>　　他抬头看向周围几人，发现诸人都摇着头，没人能回答这个问题。
<br/>　　程大雷已经不是第一次被人抓走，第一次是在青牛山，第二次是在长安，算上今天是第三次。大家都有了经验，碰见这种事倒也不如何着急。
<br/>　　只不过目前的问题是，不知道落玉寨在什么地方。上次樊梨花露面后，程大雷就派人打听过，倒是没打听到，主要是要应对十八家头领，没有在这件事上多用心。如今人家自己冒出来，还将程大雷劫走了，却是不得不查清楚。
<br/>　　“好啦，现在已经确定下来，大当家真的被劫走了。”徐神机道：“把山寨兄弟撒出去，查清这落玉寨在何处，然后想办法救大当家。”
<br/>　　李婉儿无语的看着这一幕：合着这么半天，就是确定程大雷真被劫走了。
<br/>　　距离程大雷被擒到落玉寨已经过去三天，三天内他都被关在木笼内，与骡马为伴，倒是没受刑，只是每天吃的实在是……落玉寨就没有优待俘虏这条政策么。
<br/>　　照道理来说，自己被绑过来三天，山寨的弟兄也该找过来，哪怕给人家一点银子呢，先把自己救出来再说，可为何到现在一点动静都没有。程大雷想到自己走后，山寨就是徐神机负责，那么自己到现在还没被救出来，也就理所应当了。
<br/>　　靠天靠地靠兄弟，什么时候都不如靠自己。
<br/>　　这木笼每根栅栏都有小腿粗细，凭程大雷的力气也挣不开，不过程大雷也不是没有办法。他从地上挖出一个生锈的马掌，趁着没人注意的时候一点一点的撬钉子。
<br/>　　“嘿，还活着呢。”
<br/>　　夜来，繁星漫天，樊梨花出现在程大雷面前。程大雷赶紧把马掌藏在袖子里，仰起头挤出一个笑容。
<br/>　　“樊家姑娘，我能不能给你提点意见，你们的伙食太难吃了，稍微提高一下待遇也不是不可以么？”
<br/>　　“呵，没杀你可就不错了，你还嫌饭菜难吃？”
<br/>　　“不敢，不敢，有情饮水饱么，能天天见到樊家姑娘，便是吃糠咽菜我心里也是愿意的。”
<br/>　　樊梨花这时也大概习惯了程大雷的口花花，倒也不生气。
<br/>　　“你这汉子却也有趣，陷在这里还当无事人一般，你当真不怕死么？”樊梨花脸上笑容消失，眉目变冷。
<br/>　　“牡丹花下死，做鬼也风流，若难逃一死，还请姑娘亲自动刀，方才不愧此身。”程大雷嬉皮笑脸道。
<br/>　　樊梨花低头盯着程大雷，却见他虽然衣衫狼狈，挤着一张丑脸，双眸却漆若星辰。
<br/>　　啪！
<br/>　　马鞭隔着木笼甩了进来，重重抽在程大雷脸上，留出一道伤口。
<br/>　　“哼，你也是个无用的草包，这么久了都不见你的手下来救你，怕他们都巴不得你死呢。”
<br/>　　有徐神机主事，他们就算明年才来，程大雷也不觉得稀罕。
<br/>　　樊梨花甩出一鞭，气鼓鼓的离去，程大雷扶着脸颊的伤口，悠然出神。
<br/>　　“还真是辣啊……”
<br/>　　程大雷拿出那枚马掌，继续开始撬钉子。
<br/>　　在这里关了几天，便是听周围人聊天，程大雷也大概知道些落玉寨的情况。
<br/>　　落玉寨位于帝国和戎族交界处，外面就是浩浩草原，整个落玉寨贩马为生，山谷内也有开垦出来的田地。原先的寨主名叫樊锐，死后山寨就给了自己女儿樊梨花。如今落玉寨是樊梨花以及几个叔伯主事。
<br/>　　程大雷一边撬钉子一边盼望着弟兄们来救自己。天快亮时他沉沉睡去，等第二天醒来时，发现山谷内来了一批人。
<br/>　　披头散发，穿着皮袍，骑着高头大马，腰间佩着弯刀。
<br/>　　这是戎族……
<br/>　　程大雷来了精神，难道落玉寨还和戎族有勾结。琴川关外就是戎族地界，至于属于戎族那个部落，程大雷还没有调查过。
<br/>　　这群戎族被引入聚义厅，为首之人名叫赤檑。
<br/>　　“樊寨主，吾等这次来……”
<br/>　　樊梨花插手笑道：“如果你们的来意还是和上次一样，就不必说了，我落玉寨是不会和西颜部合作的。”
<br/>　　“樊寨主误会了，我们这次来，是特意向樊寨主提亲的。”
<br/>　　“提亲？”这两个字令樊梨花楞了楞。
<br/>　　“我们首领看上了樊寨主，愿意用五百头牛为聘礼，只要樊寨主答应，以后我们两家就是一家了。”
<br/>　　樊梨花愣了好大一会儿，开口吐出一个字：“滚。”
<br/>　　“呃……”赤檑一愣：“我们都愿意拿出五百牛了，您还不满意么，要不再加二十匹马？”
<br/>　　“滚啦！”
<br/>　　程大雷不知道他们在里面谈什么，一直往这边看着。见这帮人刚进去没多久，就被赶了出来。
<br/>　　“姓樊的，你不要给脸不要脸，你真以为自己值五百头牛么。”
<br/>　　“除了戎族，我看哪有男人肯要你。”
<br/>　　一听这话，程大雷瞬间打起精神，睁大眼睛往那边看着，心里还不太明白发生了什么事。
<br/>(本章完)
