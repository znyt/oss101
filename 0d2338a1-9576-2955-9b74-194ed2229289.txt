　　在得知叶尘是散修之后，众人的态度明显冷淡了许多。
<br/>　　一路无话，大约又前行了数千里，前方出现了一座壮阔的城池。
<br/>　　“叶前辈，前面便是流光城了，是我们流光派的根基所在！”
<br/>　　云灵儿开口介绍起来。
<br/>　　叶尘点了点头，心想说，我前世还在这里生活过一年，又岂会不知？
<br/>　　来到跟前，只见城墙足足有上千米高，用硕大的黑色岩石堆砌而成，城墙之上则有一排身穿黑色盔甲的守卫，虽然也不是什么高手，但个个气势不凡，身材高大，气血旺盛，显然都修炼炼体一脉的道法。
<br/>　　而在那流光城上方，隐隐有符文流转，应该是流光派的高手，设下的护城大阵，防止外敌入侵。
<br/>　　当然了，这种程度的大阵，对于叶尘来说，根本不堪一击。
<br/>　　在云灵儿的带领下，众人在城门口落下，顿时引得那些来往的路人，纷纷侧目而视，
<br/>　　“快看！那几位好像是我们流光派的核心弟子！”
<br/>　　“那位像仙女一样美的女子，便是玄离真人座下的云灵儿师姐！”
<br/>　　“还有那位，是楚星河师兄！”
<br/>　　……
<br/>　　众人纷纷议论，个个一脸兴奋，犹如粉丝看到明星一般。
<br/>　　“叶前辈，我流光派有弟子十万，不过大多数都只是外门弟子，内门弟子只有三千，而像我们这样的核心弟子，只有数十人而已！”
<br/>　　说话之人，正是那个叫楚星河的青年男子，之前对叶尘的态度十分殷勤，在得知叶尘是散修之后，便隐隐有些轻视了。
<br/>　　说到这里，楚星河微微一顿，接着又道：
<br/>　　“至于那烈日宗，门下弟子将近百万，综合实力远在流光派之外，如果我是叶前辈的话，就会立即离开天狼星域，然后躲得远远的，永远都不要回来！”
<br/>　　楚星河说完这话，叶尘还没有开口，一旁的云灵儿早已经听不下去了，
<br/>　　“楚星河！你这话是什么意思？只要我们不说，又有谁知道今天的事情？”
<br/>　　楚星河冷冷一笑，
<br/>　　“灵儿师妹，你这话未免有些太天真了！那南黎川明显是故意找我们的麻烦，烈日宗的高层又岂会不知？如今他死了，烈日宗肯定会怀疑是我们所为，我让叶前辈离开，既是为了叶前辈，也是为了宗门着想！”
<br/>　　听到楚星河这话，叶尘不由得再次暗暗摇头，
<br/>　　他原本还想重温一下，前世在流光派的日子，顺便将烈日宗这个隐患提前除掉，如今顿时有些意兴阑珊。
<br/>　　‘反正离流光派被烈日宗灭掉，还有近十年的时间，我倒也不急于一时……’
<br/>　　一念及此，叶尘道：
<br/>　　“不知道这流光城中，可有通往其他高级星域的传送阵？”
<br/>　　楚星河还以为叶尘被自己说动，已经准备逃跑，顿时满脸喜色，
<br/>　　“当然有！就在流光宫内，等我向门中长老禀明此事，便可以送叶前辈离开！”
<br/>　　叶尘点了点头，不再多说什么。
<br/>　　一旁的云灵儿张了张嘴，想要说些什么，最终也黯然叹了一口长气。
<br/>　　众人沿着宽阔的街道，又往前走了大约十多里，前方出现了一座宏伟壮丽的宫殿，那便是流光派的总部流光宫了。
<br/>　　“叶前辈，您且在大殿稍候片刻，我这就前去请门中长老！”
<br/>　　将叶尘引入宫殿之中，楚星河撂下一句话，便直接转身离开了，其余几人也纷纷离去，唯独云灵儿留在原地，一脸的愧疚之色。
<br/>　　“叶前辈，实在不好意思啊！烈日宗势大，今日我们杀了南黎川，烈日宗肯定不会善罢甘休，您尽快离开天狼星域也好！”
<br/>　　对于云灵儿，叶尘颇有好感，微微一笑，道：
<br/>　　“我明白，倘若烈日宗真的要找你们的麻烦，你们只管将此事推到我的身上！”
<br/>　　云灵儿听到这话，顿时连忙摆手，
<br/>　　“那怎么行！叶前辈也是为了救我们，所以出手杀了南黎川和烈日宗长老，我们怎么能做那种忘恩负义的事情？”
<br/>　　叶尘淡淡一笑，没有再多说什么。
<br/>　　而与此同时，
<br/>　　后殿内，流光派掌门和诸位长老，齐聚一堂。
<br/>　　等楚星河将事情的经过一五一十说完，此时整个大殿已经炸开了锅，
<br/>　　“什么！烈日宗的少宗主南黎川，被你们杀了？”
<br/>　　“完了！这下你们可闯大祸了！”
<br/>　　“如今玄离师兄不在门中，倘若烈日宗杀上门来，我们如何抵挡的住？”
<br/>　　……
<br/>　　眼看众人越吵越凶，坐在首位上的一名中年男子，忽然开口道：
<br/>　　“星河，那斩杀南黎川之人，是什么来头，你可曾问清楚？”
<br/>　　原来此人便是流光派的现任掌门莫长林。
<br/>　　楚星河当即道：
<br/>　　“回禀掌门师尊，此人姓叶，是一名散修！”
<br/>　　“什么？散修？”
<br/>　　众人听到这话，顿时纷纷摇头，
<br/>　　散修也就意味着没有背景，除非他是化神天君级的强者，否则根本不能和烈日宗相抗衡。
<br/>　　而化神天君，是何等厉害的人物，又岂会来区区天狼星域这种小地方，插手这等小事？
<br/>　　“我听闻，但凡烈日宗嫡系弟子，自幼体内便会种下因果蛊，一旦被人所杀，凶手便会沾染上此蛊的气息，而施蛊之人，便能通过此法，来锁定凶手！”
<br/>　　一名须发尽白的老者忽然开口道。
<br/>　　“什么！”
<br/>　　众人听到这话，顿时全都脸色大变。
<br/>　　“砰！”
<br/>　　莫长林霍然站起身来，
<br/>　　“不行！此人绝对不能留在我们门中，必须尽快将他送走，以免惹祸上身！”
<br/>　　楚星河当即道：
<br/>　　“师尊，此事我已经和他达成共识，他也同意立即离开天狼星域！”
<br/>　　“太好了！我这就让人打开传送阵，送此人离开！”
<br/>　　莫长林说着，便要开始行动起来。
<br/>　　然而就在这时，
<br/>　　九天之上，忽然响起一道充满愤怒的声音，
<br/>　　“莫长林！你竟然敢纵然手下弟子，伤我儿性命，给我滚出来！出来！来！”
<br/>　　声音滚滚，犹如九天神雷，在流光宫的上方传递开来。
<br/>　　整个流光城，所有人纷纷抬头，顿时个个脸色惨然。
<br/>　　“是烈日宗！烈日宗打过来了！”
<br/>　　莫长林和楚星河等人，奔出大殿，望着天空上，那一道道强悍的气息，也纷纷脸色惨白无比。
<br/>　　“怎么会来的这么快！这可如何是好？”
<br/>(本章完)
