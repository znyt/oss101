　　第187章
<br/>　　福德勒握刀的手在抖。
<br/>　　口中呼哧呼哧喘着粗气，空气充满汗与血的味道，耳边是轰隆若惊雷的声响。血模糊了眼，周围是什么情景已看不清楚。
<br/>　　这是地狱么？
<br/>　　他扪心自问，却无人能给出回答。
<br/>　　他握刀，扬起，却不知攻击的是敌人还是同袍。
<br/>　　身边已没有多少敌人，挤满了想逃离这片无间地狱的族人。路已经被堵死了，前面的人出不去，后面更多的人向上挤，断绝了后来人的生路。
<br/>　　那么，所有人都是敌人。
<br/>　　于是，杀。
<br/>　　紧张的握刀，紧张的挥出，再握刀，再挥出，如此重复，像单纯古板的机械，或者野兽。
<br/>　　铛！
<br/>　　空气中响起干脆的刀鸣声，这是两柄刀撞在一起的声音。这样的事情也不意外，之前已经发生过无数次。
<br/>　　福德勒又一次挥刀。
<br/>　　铛！
<br/>　　随着撞击声，一只大手突然伸过来卡住了他的脖子，对方声嘶力竭的大喊。
<br/>　　“走啊。”
<br/>　　是赤眉，他的脸杵在福德勒面前，肌肉狰狞，眼睛是血红色的。
<br/>　　“走啊。”
<br/>　　这喊声令福德勒稍微清醒些，他看着完全已陷入疯狂的赤眉，表情怔了怔，下意识吐出两个字。
<br/>　　“哥哥。”
<br/>　　赤眉的刀伸过来，从福德勒的臂下穿过，扎透他身后一个蛮人。
<br/>　　“走。”
<br/>　　赤眉拽着福德勒的肩膀，硬生生的往外拖，他把刀甩出去，挂出一串血花，而又由他的臂膀一人承担。
<br/>　　“哥哥。”
<br/>　　福德勒恍然意识到一个匪夷所思的事实：赤眉在救自己。
<br/>　　踩着尸体前进，趟过由鲜血汇成的河流，手中的刀就是划过血河的桨。
<br/>　　这是一条黄泉通向人间的路。
<br/>　　城门，尸山血海。
<br/>　　金问道握紧刀立在城门外，口中大喊：“进，进，进，后退一步者，斩！”
<br/>　　城外到城内，城内到城外，哪个方向通向人间。
<br/>　　“走。”
<br/>　　赤眉将福德勒推上尸山。
<br/>　　轰
<br/>　　爆炸在其身后响起，尸体和山石飞溅，赤眉瞬间被四五具尸体压在身上。
<br/>　　“走。”
<br/>　　“哥哥。”
<br/>　　福德勒握住赤眉的手，想要将他从尸堆下拉出来。可随着人群的拥挤，赤眉身上瞬间踩了无数双脚，鲜血从他口中流出来，眼中布满血丝。
<br/>　　“走。”
<br/>　　赤眉再开口，牙龈已经开始渗血。
<br/>　　福德勒想要将自己所有力气用尽，能把赤眉拖出来，可是他所有力气并没有多少，即便用尽所有力气，依旧无能为力。
<br/>　　无能为力……多令人讨厌的字眼，而有时候无能为力就是无能为力，什么都无法改变。
<br/>　　看着福德勒脸上的表情，赤眉忽然笑了笑，用沾血的手拍拍他的头发。
<br/>　　“傻弟弟，快快长大吧……以后不要再做个废物了……快，长大吧……”
<br/>　　赤眉的声音越来越虚弱，随着一只脚踩在他的头顶，他的脸埋进血堆中，再没有抬起来。
<br/>　　福德勒被向外逃的士兵冲倒，一只脚踩着他的背走过。福德勒用刀扎进一人的腿，像攀着一棵大树站起来，他没有时间平复呼吸，用手中的刀开路。
<br/>　　已没人站在他身边，抓着他的肩膀，替他开路。接下来的路他将自己走。
<br/>　　城内，城外，接下来的路是通向人间，还是通向地狱。
<br/>　　“杀，杀，退后者死！”
<br/>　　金问道的愤怒没能阻止士兵的溃败，所谓兵败如山倒，倒山之势，绝非一人之力能阻止。
<br/>　　战场之上瞬息万变，胜者或败者可以随时转换。金问道本以为的必胜之战，却遭到程大雷迎头痛击。瓮城成了一口大锅，煮了金问道一半兵力。事后，程大雷单单清理尸体，就足足清理了十天。事后很多年，山谷外长出一种草，这种草也没什么奇怪的，只是叶脉是诡异的红色。
<br/>　　……
<br/>　　程大雷窝在山谷不出来，一心经营自己的守城之战，把山寨打造得固若金汤。因为此，程大雷对于这块大陆发生的很多事情都并不知晓。
<br/>　　由东向西，从极东的幽州海到极西之地的昆吾山，帝国的国境线像一张拉满的弓，也像一柄张开的雨伞。
<br/>　　那一年，帝国大旱，塞外的草原也遭遇大旱。草原便沙漠，牛羊成片成片的倒在，新生儿因为吃不到一口奶水，和干瘪的母亲一起饿死。
<br/>　　戎族数百个部落组成一个超大联盟，全线向帝国宣战。
<br/>　　这些部落有大有小，有强有弱，本已千疮百孔的帝国，像一个瘦弱的女孩撑着大伞走在风雨中，戎族的全线入侵就是落在伞上的狂风暴雨。
<br/>　　所以，青牛山发生的事情，并不是一件太大的事情，无非是一件大事的局部而已。
<br/>　　由东向西：程大雷紧锣密鼓的张罗守城；并州境，守军和戎族骑兵刚发生过一场遭遇战，万马奔腾，死伤无算。一名锦袍玉带，大腹便便的中年人坐在马车上狂奔，嘴里嘀嘀咕咕着：蠢蛋才守城呢，老子才不守城。
<br/>　　西北边境，一员大将苦苦支撑，翘首东盼，等着那支援的粮草。
<br/>　　京城，一留着两撇胡须的男人抿抿嘴道：“林将若是打赢了，这朝廷岂不成了他只手遮天，哼，军粮缓他一个月。”
<br/>　　江南水乡，一客商变卖家财，筹集十万担稻米北上犒军。
<br/>　　同一条路上，另外一队客商快马加鞭，正急奔京城。
<br/>　　“大家快些走，京城现在的古董字画的价钱都很低，咱们运一批回来，至少能赚三倍。”
<br/>　　南下北上，路上即有弃笔从戎的书生，也同样有逃到江南避难的达官贵人。
<br/>　　西北之地，一座孤城。一个身穿羊皮大衣的牧羊人拦在城门前，举着手中的印符，冲着溃败的逃兵大喊。
<br/>　　“我是帝国六王子，今日与此城共存亡。”
<br/>　　然后，没人理他，溃兵像流水般从他身边经过。
<br/>　　英雄，懦夫，荣耀，卑劣。
<br/>　　这片土地从不缺少逆流而上的英雄，也并不缺少唯利是图的小人，自然，更多的还是疲于奔命，努力活下来的普通人。
<br/>　　那一年，戎族大兵如潮，帝国边疆战火纷飞。青牛山则像一块洼地，当如水的兵潮流经这里时，便走不动了。
<br/>　　然后，越聚越多，越聚越多……
