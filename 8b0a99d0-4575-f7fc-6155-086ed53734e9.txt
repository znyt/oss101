　　第126章
<br/>　　“拆掉重建？”高飞虎皱起眉头道：“有这个必要嘛，咱们现在不是很好么？”
<br/>　　“好什么好？各自为战，只能被人各个击破，只有都联合起来，力量才大，拳头才狠。”
<br/>　　“大家都搬到蛤蟆寨不就行了？”高飞虎问。
<br/>　　“蛤蟆寨太小啦。”
<br/>　　程大雷叹了口气。其实择地重建山寨的想法程大雷一直都有，蛤蟆岭的地形有天然劣势，不适合以后的发展。至少，五千人住进来，蛤蟆岭就容不下。而蛤蟆岭是一块山地，可以耕作的土地太少了，想升到三级山寨有一个条件必须满足，那就是山寨拥有农田达到一千亩。
<br/>　　所以寻找一地，重新建一座山寨，是箭在弦上，不的不发的事情。
<br/>　　程大雷和高飞虎谈，就是为了山寨选址的事情，至少高飞虎对当地地形熟悉。
<br/>　　新山寨对位置，必须满足几个条件：一、要有水源，水是生命之源，没有水其他都是扯淡；二、地形上要易守难攻，这样才能保证山寨安全；三、交通要方便，要是把山寨按在山顶，易守难攻是有了，可一匹马都骑不上去，也是抓瞎；四、要有适合耕作的大量田地，在这个时代，还是农业为先，程大雷也凭空变不出粮食来。
<br/>　　偌大一座青牛山，想要找到能同时满足这四条的地方谈何容易。程大雷这几日没做其他事，把青牛山五十多家山寨都转了。所有人都对程大雷的到来表示热烈欢迎，可以说是纳头便拜。
<br/>　　当然，吃着程大雷的粮，怎么可能对程大雷不尊重。
<br/>　　这些山寨的位置也不能满足程大雷的要求，看来这些人在择地的时候，也没考虑过地理位置。
<br/>　　青牛山方圆五十里，真就找不到一个适合的地方么。
<br/>　　程大雷没有放弃，他继续带着林少羽、徐神机两个满青牛山转悠。他有一份青牛山的地形图，还是系统奖励的，按图索骥，却依旧没找到心仪的地方。
<br/>　　“大当家，我们回去吧，这大雪封山的，一脚踩不好，就掉进悬崖里了。”
<br/>　　“不回去！”程大雷伸手一指：“我们去那里！”
<br/>　　“去那儿，那么高，这要一不小心掉下来，可就摔死了。”
<br/>　　程大雷伸手所指的地方，是青牛山的主峰，名叫牛角峰，这也是整座青牛山最高的一座地方。
<br/>　　平日程大雷没想过要爬上去，今天是头一次，天寒路滑的，程大雷也加着小心。不过已经有徐神机反向毒奶过，程大雷也不是过分担心。
<br/>　　费了一番周折后，在天将黑未黑之时，三人爬上了牛角峰。登高望远，落日余晖斜照，整座青牛山的地形在程大雷眼中一览无遗。
<br/>　　整座青牛山就像一只卧倒的水牛一般，牛角冲东，牛尾冲西，一道河从牛身旁缓缓流过。
<br/>　　”紫气东来，南望京州，四蹄在水，见风则气。”徐神机感慨一声：“青牛山当真是好风水呐！”
<br/>　　“咦，军师还懂风水？”程大雷难以置信道。
<br/>　　“哈，吾仰观天文，俯察地理，至于风水小道，略懂而已。”徐神机道。
<br/>　　“咦！”程大雷情不自禁竖起大拇指：“这波装的满分！”
<br/>　　目光由东向西望，程大雷视线停留在一地，他目光所落的位置，是一座山谷。
<br/>　　“找到啦。”程大雷抛出一块石头，嘴里笑道：“紫气东来，便是王兴之相，见风则齐，是要搅动四方风云，我们要去的地方，就在那里！”
<br/>　　徐神机竖起大拇指：“大当家，你满分！”
<br/>　　“一般般啦。”
<br/>　　……
<br/>　　第二天早上，程大雷约了高飞虎、小白狼、徐神机等几人来到那座山谷。
<br/>　　整座山谷呈葫芦状，谷口能容八马并行，走进去后空间却是极大，容五千人住进去，再垦出一千亩田来不是问题。
<br/>　　山谷里长满参天大树，一人抱不过来的大树入目皆是。这个时代有一样好处，就是森林覆盖率高，老虎黑熊什么的在原始森林里一点也不稀罕。
<br/>　　山谷底部是一道穿山而来的河，当然，如今这天气河流冰冻着，今年遭了旱，河里也没有多少水。
<br/>　　易守难攻，有水源，有可开垦的农田，就是交通位置有点偏，距离官道有些远。
<br/>　　不过，世上哪有十全十美的事情，能同时满足三个条件，已经极其不错了。
<br/>　　“大当家，不能在这里建山寨啊。”徐神机突然跑过来。
<br/>　　“为何？”程大雷奇怪道。
<br/>　　“今年天旱，河里没有多少水。”徐神机道：“可明年却未必如此，等开春雪一化，老天在下几场雨，大水灌进来这里就被淹了。”
<br/>　　“这个么……”程大雷想了想，道：“却也不是太大的问题，咱们把河沟挖深些，让水排出去，正好在谷外形成一条护城河。”
<br/>　　位置定下来，接下来就是着手建设山寨的事情。幽州的冬季很漫长，这些人闲着也是无聊，正好让他们做些事情打发这个漫长的冬天。
<br/>　　首先一把火丢进山谷，如今是冬天，正应了天干物燥这四个字。大火足足烧了十天十夜，把森林化为灰烬。谷中的野兽被火熏着疯狂的往外窜，大家在谷口守株待兔，还真抓到一只老虎。
<br/>　　程大雷还是第一次吃到虎肉的味道，确实……不太好吃。
<br/>　　程大雷拥有三级山寨的图纸，只是在他脑海里，程大雷绘图不行，脑袋里有也画不出来。不过他身边有柳芷，这时代的姑娘，琴棋书画拿起来跟玩似的。
<br/>　　程大雷说，柳芷画，几番修改之后，倒是已经和程大雷脑海中的图纸差不多。
<br/>　　黄三元在此时又派上用场，担任总指挥的角色。在他的指挥下，一栋栋房屋拔地而起：医馆，粮仓，兵器库，民居……
<br/>　　所需要的材料都是把其他山寨的房屋扒了，五千人投入这件事中，工程进度却是也快。
<br/>　　自然蛤蟆寨的点将台第一时间搬了过来。
<br/>　　工程如火如荼的进行，想要彻底完工，却也不是一天两天能够搞定的。在大家都忙碌的时候，程大雷的恐惧值已经积累到十万。
<br/>
