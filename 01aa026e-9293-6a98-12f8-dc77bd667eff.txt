　　“是龙老大！”
<br/>　　众人看到龙辰出现，顿时尽皆大喜，有人已经忍不住高呼起来，
<br/>　　“龙老大，您刚刚那话是什么意思？”
<br/>　　“就是呀！龙老大，这通天塔里面，到底是什么情况，还请您给我们说说呗！”
<br/>　　云破天、绝无敌等人，顿时全都两眼发亮，纷纷望向龙辰。
<br/>　　龙辰作为目前唯一一个，从通天塔中闯关出来的人，倘若能从他的口中，事先了解一下情况，自然也就可以有备无患了。
<br/>　　龙辰微微一笑，倒也没有要隐瞒的意思，缓缓道：
<br/>　　“其实这通天塔，说简单也简单，说难也难！”
<br/>　　龙辰说到这里，故意微微一顿，所有人顿时全都竖起了耳朵，生怕落下任何一个字。
<br/>　　龙辰继续又道：
<br/>　　“通天塔的每一层，都会有一个人形傀儡守关，只要你能打败这人形傀儡，就算是过关了！”
<br/>　　“人形傀儡？”
<br/>　　所有人全都一愣，继而有人继续追问道：
<br/>　　“龙老大，那这些人形傀儡的实力如何？肯定是越往上，傀儡的实力越强喽？”
<br/>　　龙辰点了点头，道：
<br/>　　“这是自然！每一层人形傀儡的实力，都比上一层要高出一个境界，而第一层人形傀儡的实力，是和你当下的实力相当的！”
<br/>　　说到这里，龙辰生怕众人没有听明白，继续又开口解释道：
<br/>　　“譬如我眼下的修为，是元婴境六重，那第一层人形傀儡的实力，也是元婴六重！”
<br/>　　“而通天塔第二层的人形傀儡，便是元婴七重境界，第三层是元婴八重，第四层是元婴九重，第五层是元婴巅峰境界，第六层是化神一重境界……”
<br/>　　听龙辰这么一番解释，众人顿时恍然明白了过来。
<br/>　　难怪之前龙辰，到了第六层之后，速度就明显降落了下来，因为第六层的人形傀儡，已经迈入真正的化神境，实力有了一个质的飞跃，闯关的难度，自然也就开始大大增加了！
<br/>　　而洛冰玄进入通天塔的时候，刚好也是元婴六重境界，所以也是从第六层开始，难度大大增加！
<br/>　　“总的来说，这一关考验的，是我们的实战能力！越级战斗的能力越强，所能闯关的关卡，自然也就越多！”
<br/>　　最终，龙辰又总结了一句。
<br/>　　人群中经过片刻的沉思之中，众人顿时纷纷又议论开来，
<br/>　　“难怪之前规则说，修为越高，反而越不利！原来是这个意思！”
<br/>　　“龙老大是元婴六重境界，他一共闯过了十五关，也就是越了十五个境界等级，那第十五关的人形傀儡，岂不就是化神巅峰境界吗？”
<br/>　　“不错！龙老大不愧是龙老大，果然厉害啊！”
<br/>　　……
<br/>　　人群中，将众人的话全部听在耳中的叶尘，不由得眉头微微一挑，
<br/>　　“越级战斗的能力吗？这个可是我的强项！”
<br/>　　就在众人议论之间，通天塔第八层，已经开始亮起。
<br/>　　相比于龙辰，洛冰玄的前期的闯关速度，似乎要更快！
<br/>　　然而，
<br/>　　看到这一幕，叶尘却不由得暗暗摇头，
<br/>　　这一关，应该并没有时间的限制，稳扎稳打才是最佳策略，洛冰玄一味追求速度，实在是有些不智啊！
<br/>　　一晃眼，八天时间过去了，通天塔第十三层，已经亮起了近一天时间，
<br/>　　轰隆！
<br/>　　“啊~”
<br/>　　突然一道巨响落下，伴随着一道娇喘声，一道妙曼的白色身影，从塔中弹飞了出来。
<br/>　　“洛冰玄，闯关层数，十三层！”
<br/>　　“下一位！”
<br/>　　那道淡漠的声音再次响起，同时通天塔第一层的大门，也缓缓打开了。
<br/>　　众人顿时一片哗然，
<br/>　　要知道，洛冰玄在百年天骄榜首排名第二，不但实力仅次于龙辰，而且两人的差距并不算大，几乎都快可以并驾齐驱了。
<br/>　　可是如今，龙辰在通天塔中闯到了第十五层，而洛冰玄却只抵达十三层，这显然没有达到众人的预期。
<br/>　　“哈哈！诸位，我就先行一步了！”
<br/>　　云破天显然心情不错，大笑了一声之后，直接缓步迈入通天塔之中。
<br/>　　而叶尘，则第一时间来，到洛冰玄的跟前，
<br/>　　“你没事吧？”
<br/>　　他能明显感觉到，洛冰玄似乎受伤不轻。
<br/>　　不料，洛冰玄的态度，却异常的冰冷，俏脸上布满寒霜，冷冷道：
<br/>　　“我说过了，我的事情，与你无关，请你以后离我远点！”
<br/>　　说完这话，洛冰玄眼眸深处，明显露出一抹痛苦之色，撂下这句话之后，直接娇躯一晃，回到自己的修炼道场之中，开始休养调息起来。
<br/>　　洛冰玄接二连三的反常态度，顿时让叶尘越发的疑惑，
<br/>　　难道她的实力倒退，和我有关？
<br/>　　这女人，对我动了情？
<br/>　　想到这里，叶尘不由得狠狠摇了摇头，将这到荒诞的念头甩开一旁，不由得幽幽叹了一口气。
<br/>　　就在这时，忽然后方，响起了一道充满嘲讽的声音，
<br/>　　“某些人呐！不但胆大妄为，而且还癞蛤蟆想吃天鹅肉！居然痴心妄想，想要讨得冰玄仙子的欢心，真是不知死活啊！呵呵！”
<br/>　　叶尘根本不用回头，光听声音就知道，又是绝无敌那个讨人厌的家伙！
<br/>　　“原本在进入第八重星域之前，不想招惹是非的，可是你一再招惹我，真当我是好欺负的吗？”
<br/>　　叶尘眼眸中伤过一抹厉色，心中已经有了主意。
<br/>　　一转眼，
<br/>　　又过去了将近十天时间，
<br/>　　轰！
<br/>　　通天塔第十四层亮起！
<br/>　　“大家快看！云破天师兄达到第十四层了！”
<br/>　　众人纷纷惊呼起来，
<br/>　　不过并没有维持太久
<br/>　　砰！
<br/>　　第十四层亮起不到一个小时，一道人影被弹了出来。
<br/>　　“云破天，闯过层数，十四层！”
<br/>　　淡漠的声音，再次在九天之上响起，
<br/>　　“下一位！”
<br/>　　云破天从塔中弹出之后，虽然受了伤，不过却满脸喜色，虽然未能达到龙辰那样的高度，却超过了洛冰玄，也算完成他的一个夙愿。
<br/>　　接下来，
<br/>　　燕狂生、乐无忌、风华舞，先后用了差不多一个月的时间，也全都完成了闯关。
<br/>　　燕狂生止步于十三层，乐无忌和风华舞两人，则都止步于第十二层。
<br/>　　而随着风华舞，从通天塔中出来之后，接下来便轮到绝无敌了。
<br/>　　而就在绝无敌斗志昂扬，准备踏入通天塔的时候，忽然一道人影闪过，居然先他一步，进入了塔中……
