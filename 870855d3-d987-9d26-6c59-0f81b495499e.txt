　　刚回到房间。
<br/>　　“嘶”，萧逸吸了口凉气，咬了咬牙。
<br/>　　“少爷你怎么了？”依依担忧地问道。
<br/>　　“没什么。”萧逸摇了摇头，心里却暗骂，“这副身体也太弱了，才用了两下形意拳，竟然已经撑不住了。”
<br/>　　形意拳，特别是形意五绝，看似招式简单，但其实每一招都对身体有很大的要求。
<br/>　　像豹形，短时间内打出爆发性的速度，必须依靠肌肉的瞬间爆发。
<br/>　　像蛇形，正是之前萧逸躲开火烧藤的身法，如蛇乱舞，身若柔丝，快速移转腾挪，这对身体的强度要求极高。
<br/>　　如果是以前的宗师萧逸，哪怕战斗三天三夜都没事儿。
<br/>　　可现在的废物萧逸的身躯，却根本达不到要求。
<br/>　　之前萧逸看似轻松打败萧杰和萧石，其实是一直忍着身体的疼痛罢了。
<br/>　　“咔咔”萧逸扭了扭脖子，发出一阵咔咔之声。
<br/>　　那股腰酸背痛和四肢乏力让他很不舒服。
<br/>　　这副身躯虽然有着凡境一重的修为，实则只是微末修为，比普通人强不了多少。
<br/>　　“呀。”依依忽然叫了一声，道，“药快凉了，少爷赶紧趁还有余热，马上服下吧。”
<br/>　　萧逸接过碗，看着里面一团黑里黑漆的药液，微微抿了一口。
<br/>　　几乎是那口药液吞入腹中的瞬间，自己身上的酸痛立马缓解不小。
<br/>　　“好强的药力，这应该是滋补身体，温养经脉的补药吧。”萧逸笑道。
<br/>　　“嗯，这是六转养脉液呢，是三长老求大长老亲自给少爷你炼制的，极为珍贵。”依依点了点头。
<br/>　　萧逸‘嗯’了一声，看着碗里满满的药液，将碗推到了依依面前。
<br/>　　“你喝。”萧逸淡淡地说了一句。
<br/>　　“阿？”依依愣了愣。
<br/>　　“阿什么阿，让你喝就喝。”
<br/>　　“不行。”依依倔强地摇了摇头，道，“这是少爷的救命药，我不能...”
<br/>　　“我怕有毒，你先喝。”
<br/>　　“少爷放心，依依先试一试。”依依立马喝了一口。
<br/>　　半晌，她才想起来，萧逸少爷刚才不是已经微微抿了一口了吗。
<br/>　　“少爷，你...”
<br/>　　“喝完吧，我不习惯喝别人喝过的东西。你要是不想喝，倒掉便是。”
<br/>　　说罢，萧逸自顾地走到一边，摆出了一个奇怪的造型，头颈仰上，单膝独立，双手托天。
<br/>　　形意五绝之一，鹤形。
<br/>　　鹤形，没有战斗用途，却能延年鹤寿，养神疗伤。
<br/>　　以往在地球时，萧逸曾多次九死一生，绝境翻盘，每场大战之后，施展鹤形，配合内力，便能轻松治愈伤势。
<br/>　　不得不说，形意拳当真是博大精深，形意五绝更是如此。
<br/>　　要治自己如今的身体疲劳，轻而易举。
<br/>　　一旁的依依看着动作奇怪的萧逸，没敢打扰，她疑惑着。
<br/>　　疑惑今天自家少爷为什么不单止没被欺负，反而打败了萧杰萧石。
<br/>　　疑惑萧逸少爷今天为什么不拿她出气，不打她了。
<br/>　　疑惑萧逸少爷为何今天对她这么好，将珍贵的药液给她喝。
<br/>　　她有些天然呆，但不是笨蛋，自然明白萧逸是故意要将药液给她。
<br/>　　她想不明白，但没有再多想，而是听话地将所有药液喝光。
<br/>　　喝得一滴不剩。
<br/>　　萧逸给的东西，她珍如重宝。
<br/>　　几乎是肉眼可见的，依依原本苍白的脸色稍稍恢复了些许红润；脸色的淤青伤痕也褪了一些。
<br/>　　一旁的萧逸一边维持着鹤形，一边瞥了一眼依依，淡淡地笑了笑。
<br/>　　数分钟后，萧逸收回了奇怪的动作，身上的酸痛和乏力已经消失，整个人神清气爽，浑身有劲。
<br/>　　他看了眼依依，有些心疼。
<br/>　　依依脸部的轮廓很精致，像一件精致地艺术品般完美，妥妥的美人胚子。如果不是长期受到虐待和打骂，现在必然是个绝世美女吧。
<br/>　　那乖巧懂事的性格，更是让人心生怜爱。
<br/>　　“下次有危险，不要再挡在我面前了。”萧逸想起之前依依的举动，嘱咐了一句。
<br/>　　要不是他及时出手，以依依的孱弱身体，以萧杰的力度，依依的手臂绝对会伤得很严重。
<br/>　　“依依的命是少爷的，依依...”
<br/>　　“我不需要别人的保护。”萧逸冷酷地打断道。
<br/>　　“还有，别再说你的命是我的，我不习惯身上背着别人的命。”
<br/>　　作为一个杀手，萧逸孤独惯了，对于命，他只有两个概念，要么是自己的命，要么是别人的命；要么是能杀的命，要么是与自己毫无关联的命。
<br/>　　“嗯。”依依乖巧地点了点头，道，“依依的命是少爷的，一切依少爷的。”
<br/>　　“你...”萧逸皱了皱眉头。
<br/>　　“什么都听我的是吧？”萧逸笑了笑。
<br/>　　“嗯。”依依点头。
<br/>　　“好。”萧逸道，“要么以后别再让我听到‘你的命是我的’这种蠢话；要么，你就在这句话前加上萧逸少爷风流倜傥英俊潇洒玉树临风绝世无双帅到掉渣，依依什么都听少爷的。”
<br/>　　“明白了吗？”萧逸在为难依依。
<br/>　　“额。”依依愣了愣，道，“依依听少爷的，只不过，少爷能不能再说一次，说慢点儿，依依记不住。”
<br/>　　“记不住是你的事。”萧逸淡淡道。
<br/>　　“可是...”
<br/>　　“怎么，不听我的话吗？”萧逸皱眉道。
<br/>　　“听。”依依点了点头，而后低下了脑袋，似乎感到很为难。
<br/>　　萧逸没有理会她，随手拿起了之前三长老留下的那个袋子。
<br/>　　袋子里，正是他这个月的修炼资源。
<br/>　　萧逸打开来看了看，里面有二十两散银，还有一个玉瓶。
<br/>　　将玉瓶打开，顿时一股丹香扑面而来。
<br/>　　瓶中，有三颗晶莹剔透的丹药。
<br/>　　这是淬体丹，一品丹药。
<br/>　　凡境武者主要还是淬炼身体，打好基础，日后才能追求更高的境界。
<br/>　　而说到丹药，就不得不提炼药师。
<br/>　　这是一个在炎龙大陆中稀罕而尊贵的职业。
<br/>　　要成为炼药师，武者的资质和天赋必须极高，其次，对其他各项要求也极为严苛。
<br/>　　可以说，一百个武者中，也难找到一个炼药师。
<br/>　　而炼药师的尊贵，就在于他们能炼制出各种神奇的丹药。修炼的丹药、疗伤的丹药、炼体的丹药等等....
<br/>　　而炼药师，又分九个等级。从低到高，分别是一品炼药师、二品炼药师....九品炼药师。
<br/>　　传说，九品之上，还有皇品炼药师，只不过没人见过就是了。
<br/>　　萧家大长老，萧离火，除了是一位强大的先天境武者外，还是一位尊贵的三品炼药师。
<br/>　　家族中的丹药，大多是他炼制而成。
<br/>　　整个紫云城，也不过只有三个三品炼药师罢了，分别在萧家、慕容家和江家，可想而知炼药师的稀有。
<br/>　　“淬体丹。”萧逸拿了一粒，吞入口中。
<br/>　　同时，发动体内控火兽武魂。
<br/>　　顿时，丹药化为阵阵精纯药力，游走于全身，一部分淬炼肉身，一部分被吸纳于丹田。
<br/>　　只不过，速度很慢。
<br/>　　按这速度，起码要一天才能吸收完这颗淬体丹的全部药力。
<br/>　　“控火兽武魂，果然很渣。”萧逸无奈地摇了摇头。
<br/>　　武魂的等级差别，就在于吸收天地灵气，以及吸收丹药的速度。这种速度，也就造成了武者的修炼快慢，从而有了天才和废物。
<br/>　　像家族中的橙阶武魂子弟，半天就能吸收完淬体丹。
<br/>　　撇去丹药，就拿日常的修炼来说。橙阶武魂拥有着，修炼五天，顶得上赤阶武魂的十天。
<br/>　　两者足足差了两倍。
<br/>　　萧逸毫不犹豫停下了控火兽武魂，转而使用冰鸾剑武魂。
<br/>　　嗖，几乎是在武魂发动的瞬间，淬体丹顷刻消融，全部药力转化为体内真气。
<br/>　　“好恐怖的速度，真不愧是最顶尖的紫色武魂。”萧逸心里吃了一惊。
<br/>　　如法炮制，萧逸将另外两颗淬体丹也吞掉。
<br/>　　数秒后，体内‘腾’地一声，一股莫名的强大力量爆发。
<br/>　　这是突破的象征。
<br/>　　“突破到凡境二重了。”萧逸露出了笑意。
<br/>　　同时，萧逸明显可以感觉到，体内丹田已经能调动出真气。
<br/>　　真气，是武者体内的力量，武者发动攻击，少不了真气。
<br/>　　萧逸试了试，催动真气，顿时，一股爆发性的力量凝聚于拳头。
<br/>　　“好强，真气，怕是比地球的内力要强上一些。”萧逸做了下判断。
<br/>　　“可惜，毕竟只有凡境二重，真气的量太少了。”萧逸摇了摇头。
<br/>　　武者修炼，有九层境界，从低到高，分别是凡境、后天、先天、洞玄、破玄、地元、天元、地极、天极。
<br/>　　传说，踏破天极之后，便是武神之境。
<br/>　　武神，乃是武者最高的境界。遨游太虚、掌控天地、无所不能。
<br/>　　挥手间，破碎山河；踏踏脚，天崩地裂；一己之力，足以主宰亿万生灵。
<br/>　　当然，武神距离现在的萧逸还太遥远了。
<br/>　　哪怕是整个萧家，最强者也不过是九大先天境长老。
<br/>　　先天之后，洞玄境的武者，已经能飞天遁地，手段莫测。这种层次的武者，一向神龙见首不见尾，整个紫云城都不存在。
<br/>　　“现在想那么多也没用，还是抓紧修炼吧。”萧逸摇摇头，甩去心中杂念。
<br/>　　在紫色武魂冰鸾剑的发动下，天地灵气如河流汇入大海般，澎湃地朝他体内涌去。
<br/>　　体内真气越来越多，丹田也越加凝实强大。
