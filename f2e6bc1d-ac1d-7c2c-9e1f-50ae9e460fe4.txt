<br/>    一念万剑生，剑剑不相同。
<br/>    
<br/>        国器十二剑，代表着八极的八柄剑分明叫做丛云，朔风，惊雷，天光，厚土，善水，炎音，煌金。悬于天空之中的那柄剑，名字则是叫做天诛，地面之上的那一把，叫做浊坤，穿梭在虚空之中的那一柄剑，名叫虚弥。
<br/>    
<br/>        陈明从这十二柄剑之中一一走过，反正现在有着风妖老祖守护着燕山，其他古仙门和他又是同盟，这剑峰上的剑，就全部炼化到万道天竹之中吧。
<br/>    
<br/>        说实话，这十二并剑，陈明现在连一柄的威力都发挥不出来，毕竟他现在的法力对于国器来说，还是差距太大了，不过用国器肯定比以前的那八柄灵剑要好得多。
<br/>    
<br/>        把国器当灵兵用，恐怕也就只有陈明了。
<br/>    
<br/>        不过将这些国器全部炼化到万道天竹之中是个耗时间的事，陈明现在也没空，就先让这座大阵在这儿吧。
<br/>    
<br/>        而且，就是现在想要带走，也不知道怎么操控这个阵法，停下阵法啊。
<br/>    
<br/>        等到将燕山全部迁过来，再来慢慢弄这些事。
<br/>    
<br/>        陈明本想在这里找一找，看看有没有灵脉，像是御剑宗这样强大的宗门，灵脉应该是有的的，但是实际上，这些灵脉因为没人照看，加上风妖肆虐的缘故，只剩下了两条风属性灵脉孕育的灵魅还活着。
<br/>    
<br/>        陈明找了一圈，找到了两个浑身细小羽毛的小人，看到陈明之后一阵惊慌，直接抱着几颗不知名的种子跑到灵脉之中去了。
<br/>    
<br/>        看来时间过得太久，这些灵魅都已经快要失去关于人的记忆了。
<br/>    
<br/>        看起来，倒是像是两只小鸽子，陈明在储物戒中寻找了一番，拿出一只鸡，手中生火，在两只小鸽子面前烤的通体金黄，金黄色的油脂飘香，陈明撕下一块鸡肉尝了尝，已经熟了，扯下几根肉丝，缓缓的递了过去。
<br/>    
<br/>        小鸽子明显还是害怕，但是闻着鸡肉的香味，终于是抵抗不住八百年没有吃肉的诱惑，每只小鸽子接过，开始往小嘴里疯狂塞，吃完了两根肉丝，两只小鸽子眨着大眼睛对陈明进行了成吨的卖萌攻击，陈明一边撕下肉丝，一边悄悄画好了灵契，直接将两个小家伙契约，成功得到了两条灵脉。
<br/>    
<br/>        两个小家伙一左一右坐在陈明的肩头，陈明也是跟着卓青瑶他们去看看御剑宗留下的功法术法，陈灵鱼见到两个可爱的小鸽子，伸手直接拎过去一只，在怀里蹂躏。
<br/>    
<br/>        来到御剑宗的藏经阁，陈明看向这密密麻麻的书架，书架上面厚厚的一层，都是灰。
<br/>    
<br/>        陈明伸手拿起一本，经文在入手的瞬间就已经腐朽了，化作一手的灰尘，岁月过了太久，都已经无法保存了。
<br/>    
<br/>        看到四个小家伙跃跃欲试的样子，陈明赶紧挥手道：“停手！”
<br/>    
<br/>        然后陈明就看到，十三本经文已经在四个小家伙的手中化成飞灰了。
<br/>    
<br/>        要是这其中有着控制万剑诛心大阵的经文，我非抽死你们几个。
<br/>    
<br/>        凌仙道：“师父，这些都是普通的书籍，过了太久，已经腐朽了。”
<br/>    
<br/>        陈明冷笑一声，你陈仙师什么办不到，系统小姐姐的黑科技可是层出不穷，丹鼎仙人的笔迹她都能认出来，还怕这些？“放着，每一本经文都在自己讲述大道，我可以听到它们的声音。”
<br/>    
<br/>        李素衣惊道：“这也行？”
<br/>    
<br/>        陈明道：“你们在里面，太容易破坏到这些经文了，你们在外面等我。”
<br/>    
<br/>        让四个小家伙都出去，陈明关上门，拿起一盏青铜灯，轻手轻脚行走着，光是御剑术，陈明就见到了两百多种，仙级功法陈明倒是没有见过，但是能够修行到大君的功法，陈明倒是看到了九种。
<br/>    
<br/>        至于能够修行到王级的功法，足足不下百种。
<br/>    
<br/>        即便是陈明看到，也是心动不已，上百种王级功法，又是上百枚道种，到时候，有那种修为，仙术当成鞭炮炸！
<br/>    
<br/>        随手不小心就是一个仙术炸下去，那感觉，应该是相当的不错的。
<br/>    
<br/>        在其中小心看了一阵，看见一本经文便询问系统小姐姐一次，“顿悟这本经文需要多少师德？”
<br/>    
<br/>        陈明靠着系统小姐姐，也是知道了这些经文的作用，终于是找到了万剑诛心大阵的经文，其实这阵法本来的名字并不叫万剑诛心，万剑诛心只是其他人对于这阵法的评价，这阵法真正的名字，叫做玄关剑舞。
<br/>    
<br/>        而且，这阵法准确地来说，并不能只是称之为阵法了，这更像是，一种驾驭飞剑之法，不过是统一协调的飞剑，而且还是构成大阵的飞剑。
<br/>    
<br/>        “叮，此阵法分为两卷，天字卷与地字卷，领悟地字卷，可成为地级阵法师大成，共需要三万点师德，天字卷共需要三十万师德。”
<br/>    
<br/>        这需要的师德也太狂暴了，要不是现在有着这么多的记名弟子，还真是一个大问题。
<br/>    
<br/>        一层需要的时间也不多,静静地等待三个月就好，而且随着入道境的弟子越来越多，陈明目前需要做的事情也越来越多了，比如给入道境的弟子讲道，给入道境的弟子炼制丹药，并且写成书册，供他们自己学习。
<br/>    
<br/>        御剑宗这个地方，比原来的燕山好多了，炼丹，炼器的工具，炉鼎，都很齐全，这些毕竟都是灵兵，几百年还不至于腐朽，还能够接着用。
<br/>    
<br/>        而且御剑宗这个地方，也足够大，至少比现在燕山境内大多了，陈明估计，应该是燕山全境的十倍大，最重要的是安全，而且自己在这里炼化剑峰上的灵兵，十二国器，也不好照顾到燕山，来回跑太麻烦了。
<br/>    
<br/>        带着四个亲传弟子找到等候他们的血衣剑卫，一行人回到燕山，陈明立即召集了燕山的要员，六位妖王齐聚，四大宗主在场，如今黑虎和火燕已经快要达到神府境，修为精进了不少，白骨饕餮，不愧是大杀器。
<br/>    
<br/>        陈明先是询问了一番妖军的进度，黑虎禀报道：“山主，如今我们收编了所有妖王大部分的妖军，血衣剑卫的数量已经扩充到了三万，燕山的镖师也已经扩充到了一万，所有的妖军，已经全部落入了我们的手中！”
