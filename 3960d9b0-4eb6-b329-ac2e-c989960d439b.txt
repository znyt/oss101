　　第649章
<br/>　　百里家族的没落，也并非一日两日。传到这一代，在帝国的世家豪族中，已经摆不上桌面。最后，家族出了百里胜这样一位。才干武学，人情事故都有过人之处。
<br/>　　百里家族倾整个家族之力培养他，将他送到帝国侯爷的位置。百里胜果然也没有辜负家族的期望，攀上将军府这棵大树，帝国朝堂间的勾心斗角他也玩得转。慢慢，百里胜这个没落的家族，有了些起死回生的味道。
<br/>　　百里无常盯着百里胜，依旧沉着一张脸，让人看不透他心中想些什么。但今日他的目光，隐隐有了杀意。
<br/>　　“七弟，你站得太高，诸多事你看不到。家族倾力培养你，但你这棵大树遮蔽了太多人。我们，或许并不比你弱，但你却拿走了本该属于我们的东西。”
<br/>　　百里胜这才想起些什么，自己与这位三哥从小一起长大，但关系绝对谈不上好。似乎，在整个家族中，自己也并没有可以交心的存在。
<br/>　　“我当是怎么会事，却原来是因为那些童年小事，三哥，有些仇你记的时间太长了吧。”
<br/>　　百里无常又一次摇摇头：“我刚才已经说了，你看得太高了，看不清楚你脚底下的人是怎么活着。对于一个家族来说，延续下去比什么都重要，家族可以牺牲你，当然也可以随时牺牲我，只是，这次家族选择牺牲的是你。”
<br/>　　“我被家族牺牲了？”
<br/>　　“尉迟将军行将就木，将军府怕已经不是遮荫的大树。”
<br/>　　“所以……家族投靠了相府？”
<br/>　　“想要投靠相府，总是需要一份投名状。”百里无常道：“这次，家族需要你的牺牲。”
<br/>　　“真没想到，我这颗人头临死前还有些价值。”百里胜昂起头：“现在我这颗人头就在这里，有本事的你过来取。”
<br/>　　百里胜亮出双鞭，其他人的兵器都被百里无常变着法子收走，他却是还随身带着。
<br/>　　百里无常看了一眼靠在身后的大刀，轻轻摇了摇头，挥手道：“杀，一个不留。”
<br/>　　三水关的守军一拥而上，手中长矛将八百残兵团团围住，不留一点空隙。他们这些人醉的醉，伤的伤，而且手无寸铁，根本没有力气抵抗。短暂瞬间，便有数人被杀。
<br/>　　人群以百里胜为中心聚集，他倒是想和百里无常单挑较量，看看个人武艺如何。但百里无常根本没有与他单挑的意思，他一挥手，兵将便一拥而上，怎有人能够活命。
<br/>　　隔着人群，百里胜看到那张死气沉沉的脸。他恍然想起什么，自己这位三哥，向来是不见兔子不撒鹰，不做没把握的事。
<br/>　　他敢设下今日这场局，便是料定自己无法逃生。百里胜长舒一口气，此时此地，自己已经没有别的办法，唯有拼死一战。
<br/>　　好在，这样的绝地，百里胜也并不陌生。他擎起钢鞭，口中喝了一声，带着人向前冲杀。
<br/>　　这八百残兵都是从刀山火海中走出来的，能够只生死战中活到现在，每个人多少都有些机灵劲儿。
<br/>　　大家就地取材，用随手拿得到的东西做兵器，有些将敌人的兵器夺过来，府中地方狭窄，一下涌入如此多的人，有些施展不开。
<br/>　　靠此，百里胜的部下竟然渐渐稳住了局势。
<br/>　　百里无常看着这一幕，眉头皱了起来。自己这位七弟，果然是有些非凡之处。当然，能撑着走到现在，百里胜也算是不简单了。
<br/>　　他终于还是握住了自己的兵器，向前跨出一步，大刀劈了出去。
<br/>　　铛地一声，百里胜以双鞭接住大刀，他摆摆手，道：“来。”
<br/>　　二人厮杀在一起，百里胜早已身心俱疲，但靠着求生的决绝意志，竟然能与百里无常厮杀不落下风。
<br/>　　百里无常小看了自己这位七弟，但百里胜同样小看了对方。在族中，百里无常虽然身材魁梧，擅使大刀，但却是一位智将，并不以刀马武艺见长。百里胜没想到，他用刀的功夫也如此了得。
<br/>　　久战吃亏，而自己就算胜过百里无常又能如何，依旧逃不过这死局。
<br/>　　正这时候，有一员武将向前跨出一步，他赤手空拳，竟从百里无常的刀法破绽中挤了出去。单手卡住百里无常的脖子，猛地将他摔在地上，另外一只手夺过了他的大刀。
<br/>　　百里无常本人还没搞明白是怎样情形，全身骨骼便如同被摔断相仿。他刚抬起头，百里胜的钢鞭已经架在他脖子上。
<br/>　　“说，要死要活？”百里胜冷声喝道。
<br/>　　刚才出手的自然就是卢俊义，他一直没有出声，寻找最佳机会出手，果然收到奇效。
<br/>　　这一局本无破绽，但没想到，百里胜手下还有这样的悍将。他本身武艺并不弱，对方却可以一招将自己击败，很明显，他的武艺远在自己身上。
<br/>　　“我放你们出关。”
<br/>　　百里胜竟有几分哭笑不得：“三哥呐三哥，你果然认得清局势，取我们的兵器马匹来。”
<br/>　　百里无常被卢俊义押着动弹不得，他挥挥手，命令手下照办。
<br/>　　残兵取过兵器马匹，列队出关，他们在三水关中带足干粮，押着百里无常出了三水关。
<br/>　　过了三水关，便是京州境内，从这里到长安已经没有多少路程。
<br/>　　三水关的兵丁聚在城门前，因为百里无常在对方手中扣着，所以不敢上前。
<br/>　　“侯爷，这个人该怎么处理？”卢俊义问。
<br/>　　百里胜盯着百里无常一张脸，都到现在这个情形，他脸上仍然没有多少表情。
<br/>　　“三哥，如果为死去的弟兄想，我不能让你活着。可如果想想我们兄弟情分，我也不能让你死。”百里胜无可奈何的摇摇头：“怎么选都是错，怎么选都不对。”
<br/>　　他的钢鞭突然砸在对方胳膊上，百里无常再也忍不住，发出一声痛呼。
<br/>　　“要你一条胳膊，算是为死去的弟兄。如果我不死，日后我们还会有见面的机会。”百里胜顿了顿：“下次见面，我还唤你一声三哥。”
<br/>　　百里胜深吸一口气，道：“放开他，我们走。”
<br/>(本章完)
