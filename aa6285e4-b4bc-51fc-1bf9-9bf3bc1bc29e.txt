　　“堂堂暗影楼少楼主，竟然亲自出动，还真是让易某意外。”
<br/>　　萧逸淡淡地说道。
<br/>　　“笑话。”钟无忧冷声道，“你以为一群区区山贼首领能请动本公子？”
<br/>　　“不过是恰好在附近，过来看看罢了。”
<br/>　　钟无忧说着，桀骜的双目看向天残地缺四人的尸体，冷声道，“没想到过来看一眼，却见到了四个废物。”
<br/>　　说罢，又看向萧逸，冷声道，“还遇到了你，最近声名鹊起的天才猎妖师，易霄。”
<br/>　　钟无忧冷酷的双眸盯着萧逸，仿佛要看穿萧逸。
<br/>　　萧逸也是这般看着他，直视他的双眸。
<br/>　　萧逸曾经听说过钟无忧的名字，暗影楼少楼主钟无忧，他自然是知道的，只不过却知之甚少，更是第一次见到真人。
<br/>　　钟无忧的气息，只是半步洞玄修为。
<br/>　　但，钟无忧给他的感觉，却比天残地缺和黑白无常四人加起来还要浓烈，压迫感更强。
<br/>　　两双同样锋芒毕露的锐利双目互相对视着，周遭的空气，竟在此刻产生了一股难以言喻的肃杀之息。
<br/>　　半晌，钟无忧率先收回了目光，冷声道，“那四人虽然是废物，但始终是我暗影楼的人，你杀了他们，便把命留下来。给你半个时辰，将真气恢复好。”
<br/>　　“不必。”萧逸淡淡地摇摇头。
<br/>　　钟无忧霸道地说道，“你恢复也好，不恢复也罢。总之，半个时辰后，本公子才会杀你。”
<br/>　　萧逸淡淡道，“你若要战，易某奉陪。你若不战，我也懒得浪费时间。”
<br/>　　说罢，萧逸转身便欲离去。
<br/>　　“站住。”钟无忧冷喝了一声，“易霄，在这北山郡，没人敢忤逆本公子的话。”
<br/>　　说罢，钟无忧大手一挥。
<br/>　　顿时，地面剧烈地震动起来。
<br/>　　方圆千米之内，大地龟裂，山石崩塌。
<br/>　　特别是在萧逸旁边，一颗参天之树迅速从地底下冒出，高达百米，枝叶遮天蔽日。
<br/>　　连绵不断的须根，囊括千米。
<br/>　　嗖嗖嗖，无数须根挥扬。
<br/>　　千米范围之内，无疑成了一个树藤飞扬的诡异之地。
<br/>　　倏忽间，无数须根又互相融合，成为九条粗壮无比的须柱。
<br/>　　须柱数十米粗，长约数百米，肆意摆动，如同九条灰色长龙，将前方之路完全阻隔。
<br/>　　钟无忧冷声道，“如今千米范围之内，尽在本公子的掌控之中。你最好老实地待着，否则，这无数须根，会将你绞成碎片。”
<br/>　　“呵。”萧逸怡然不惧，冷声道，“钟无忧，你似乎忘了我的手段。你这九条大虫子，还不够我一阵烧的。”
<br/>　　......
<br/>　　百武城，是这附近十数城的最强之城。
<br/>　　在它周围，还有一些稍弱的城池。
<br/>　　其中一城，名为柳絮城，正是萧逸之前在茶馆歇息，并离开的城池。
<br/>　　此时，城内忽然一阵震动，弄得人心惶惶，不少武者更是飞跃而出，想要探个究竟。
<br/>　　剧烈震动的源头，来源于城外。
<br/>　　武者、猎妖师、城内一些家族的强者，纷纷往城外跃去。
<br/>　　“如此剧烈的震动，难道是有某只妖兽在作乱？”
<br/>　　“该死，速速去将那妖兽击杀，可不能让它到柳絮城来杀人。”
<br/>　　猎妖师们一边往城外跑着，一边说道。
<br/>　　“不，不像是妖兽。你们看，城外高空中，似有一片巨大的绿色。”
<br/>　　“此乃异象，难道是有重宝出世？”
<br/>　　一些家族武者惊讶地说着，加快了往城外赶去的速度。
<br/>　　.....
<br/>　　之前萧逸与天残地缺、黑白无常战斗的地方，本就距离柳絮城不远。
<br/>　　毕竟他刚出城没多远就被他们四人拦下了。
<br/>　　五大半步洞玄武者的战斗，威势可不小，已经让城内武者有所察觉了。
<br/>　　如今钟无忧弄出如此大的阵势，千米之地震荡不已，连柳絮城都被波及到，城内各大武者自然也惊动了。
<br/>　　.....
<br/>　　不多时，数百武者已经赶到城外。
<br/>　　有的是猎妖师，有的是独行武者，有的是城内各大家族的人。
<br/>　　“好巨大的树。”
<br/>　　“你们看，树下那四具尸体怎么有些眼熟？”
<br/>　　武者们没敢靠近，小心翼翼地眺目远视着，议论纷纷。
<br/>　　“那服饰还有面具上的‘影’字，是暗影楼的人。对了，他们四人不是天残地缺和黑白无常吗？”
<br/>　　“那四个凶名赫赫的杀手怎么死了？我记得他们可是半步洞玄的修为，谁有那个本事杀得了他们。”
<br/>　　“这些年被他们暗杀的武者和猎妖师可不在少数，也不见谁能奈何得了他们。”
<br/>　　各个武者们惊讶地说着。
<br/>　　半晌，一个猎妖师惊声道，“那个家伙，一声黑袍，戴着面具，难道是易霄。”
<br/>　　“易霄？”
<br/>　　此言一出，周围的猎妖师眼睛一亮。
<br/>　　“我想起来了，最近易霄横扫各大山贼团伙。那些山贼首领出了高价请动天残地缺他们追杀易霄。”
<br/>　　“看来，是天残地缺他们没能杀掉易霄，反倒被易霄击杀了。”
<br/>　　“啧啧，那易霄不愧是天才猎妖师。天残地缺他们可是成名已久的杀手了，竟然被他给击杀了。”
<br/>　　“这等天才，日后必然又是我们北山郡名动一方的强者。”
<br/>　　猎妖师们的赞叹之声此起彼伏。
<br/>　　不过，还是有不少武者面露疑惑，道，“那易霄对面的人是谁阿？看起来年纪也不大，顶多17、8岁。我怎么感觉他很强。”
<br/>　　在场的武者都皱起了眉头。
<br/>　　他们能认出易霄，是因为易霄最近在不少城池都出现过。而且一路横扫，很多猎妖师也都见过他。
<br/>　　钟无忧，虽然名声更显，但他那种人物，可不是一般人能见到的。
<br/>　　半晌，还是一个家族强者忽然脸色大变，认出了钟无忧。
<br/>　　“易霄对面之人...是...是无忧公子。”这个家族强者声音都变得颤抖起来。
<br/>　　“无忧公子？”一众武者先是一愣，而后同样脸色大变。
<br/>　　众人的目光，忽然齐刷刷地看向前方的巨大之树。
<br/>　　“果然是无忧公子，如果没有猜错的话，那就是参天树。”
<br/>　　“没错，那是青色武魂，参天树。”
<br/>　　“当年暗影楼传出无忧公子觉醒了青色武魂时，可是让整个北山郡各大势力都为之震惊。”
<br/>　　众人看向钟无忧的脸色，比刚才看到易霄时更为惊骇。
<br/>　　“那易霄怕是有麻烦了。”
<br/>　　“无忧公子除了是暗影楼少楼主外，更是北山榜第十的超级天才，小小年纪已是半步洞玄修为。”
<br/>　　“曾有传言，他还斩杀过一个洞玄境武者。”
<br/>　　北山榜，代表着北山郡最出色的年轻一代。
<br/>　　上榜者，无不是声名远播的青年才俊。
<br/>　　排名从一到五十，越是靠前，实力越强，天赋越高。
<br/>　　排名前十者，更个个都是超级天才。
<br/>　　北山郡的成名强者里，几乎每一个年轻时都是榜单上的人物。
<br/>　　可想而知这个榜单的含金量有多高。
<br/>　　北山榜的排名，并非固定不变。
<br/>　　这钟无忧虽然如今排名第十，但他却是前十名中年纪最小之人。
<br/>　　据传闻，以他的天赋，不出数年必能挤进前五。
<br/>　　暗影楼楼主更是断言，数年之内，他就能突破洞玄境，三十岁前便能突破破玄境，执掌整个暗影楼。
<br/>　　可想而知，这钟无忧在北山郡是何等天才。
<br/>　　也难怪在场的武者在认出他时，个个脸色大变。
<br/>　　......
<br/>　　萧逸与钟无忧这边。
<br/>　　漫天火海剧烈地燃烧着这棵参天大树，那九条须柱如同九条灰色长龙，快速朝萧逸袭去。
<br/>　　萧逸眉头一皱，脚步飞跃，快速躲开。
<br/>　　轰、轰、轰、、、
<br/>　　灰色长龙重重地砸在地面上，将地面砸出九个巨型坑洞。
<br/>　　坑洞内，满地齑粉。
<br/>　　萧逸脸色一凛，暗道，“每条须柱足有数十米粗，数百米长，如此重的力量砸下来，将地面都砸成齑粉，若是武者的区区肉身被拍中了，不死也残。”
<br/>　　“易霄，我警告过你了，最好不要乱动，安静地等半个时辰，否则，你会提前死去。”钟无忧冰冷而霸道地说道。
<br/>　　“哼。”萧逸冷哼一声，身上的普通火焰变了。
<br/>　　漫天紫炎澎湃而出。
<br/>　　一股妖异的气息充斥在空气中。
<br/>　　嘶...嘶...嘶.....
<br/>　　九条须柱甚至没能反应过来，顷刻间便被烧成了灰屑。
<br/>　　而后，紫炎不灭，散落在千米范围内的其他须根，以及参天之树上。
<br/>　　紫炎在须根树叶之中燃烧，密布在参天树的任何一个角落。
<br/>　　远远看去，这片千米范围，变成了一个紫炎纵横的火焰世界。
<br/>　　“嗯？”钟无忧眉头一皱，不可思议地看着自己的参天树在快速被烧毁。
<br/>　　萧逸冷声道，“钟无忧，怕是你那半个时辰，等不下去了。再等下去，就等着你的武魂被烧成虚无吧。”
<br/>　　“你...”钟无忧脸色一变，满是杀气地盯着萧逸。
