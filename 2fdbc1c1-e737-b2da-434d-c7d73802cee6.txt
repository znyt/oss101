　　猎妖殿主殿，最后方。
<br/>　　有一座紧挨着殿主居所的巨大建筑。
<br/>　　顺带一提，整个王都，面积极大，不亚于一个拥有二百城以上的郡。
<br/>　　故，里头的大势力，宗门或山门，占地极其庞大。
<br/>　　如猎妖殿主殿，看似只是一座建筑，实则占地面积不亚于一座大城。
<br/>　　里头楼阁、殿宇之类的建筑，多不胜数。
<br/>　　猎妖殿主殿，毕竟要统领全炎武王国三百六十分殿。
<br/>　　每日来此办事的猎妖师，发布任务的武者等等，也多不胜数。
<br/>　　当然了，猎妖殿主殿可分两部分。
<br/>　　前方部分，是殿内工作人员的办事地点。
<br/>　　任何人都可行走于此。
<br/>　　后方部分，则是殿内人员的休息之所。
<br/>　　除却猎妖殿主殿人员，任何人等，不得随意进入。
<br/>　　卷宗室、猎妖殿宝库等重要地方，都在后方部分。
<br/>　　殿主的办公房间、以及少数一些分殿主的办公房间，也在后方部分。
<br/>　　言归正传。
<br/>　　此时，殿主与萧逸二人，正站在主殿最后方的那座巨大建筑前。
<br/>　　“传承殿。”萧逸看了眼巨大建筑中央的牌匾，自语一声。
<br/>　　不错，这里是传承殿。
<br/>　　猎妖殿无数年来的传承，皆在此处。
<br/>　　殿主带萧逸来此，自然是要接受传承的。
<br/>　　萧逸迟疑了一下，道，“殿主，那场盛事，猎妖殿不能派别人去参加吗？”
<br/>　　“怎么？”殿主问道，“你不想去，或者有事缠身？”
<br/>　　“倒不是。”萧逸摇摇头，道，“只是小子觉得，猎妖殿有大把的天骄和强者。”
<br/>　　“小子毕竟资历尚浅，或许殿主可以将重任委以他人。”
<br/>　　“就这个原因？”殿主问道。
<br/>　　“嗯。”萧逸点点头。
<br/>　　“那便不用废话了。”殿主大手一挥，传承殿的保护禁制瞬间被打开。
<br/>　　随即，一股庞大的天地之力，禁锢了萧逸。
<br/>　　以萧逸现今实力，竟是动弹不得。
<br/>　　殿主再次大手一挥，已将萧逸轰入传承殿内。
<br/>　　萧逸一个趔趄，差点没摔倒。
<br/>　　‘嘭’的一声。
<br/>　　传承殿大门被关上。
<br/>　　原地只留下一句话，“好好参悟，若是参悟失败，便不要出来了。”
<br/>　　此时，传承殿的禁制再次被触发。
<br/>　　任何人进不得，出不得。
<br/>　　萧逸站稳身子，苦笑一声。
<br/>　　随即，目光打量着传承殿。
<br/>　　率先映入眼帘的，是一个个灵位，密密麻麻，不下数千。
<br/>　　每一个灵位，都是属于曾经的猎妖殿先辈。
<br/>　　上面的每一个名字，皆是曾经名震大陆的强者。
<br/>　　每一个灵位，都被一股淡绿色光芒所包裹。
<br/>　　萧逸感知了一番，顿时脸色一惊，“好恐怖的力量。”
<br/>　　每一个灵位上的光芒，都充斥着庞大的力量。
<br/>　　那些，便是属于这位前辈的传承。
<br/>　　猎妖殿的传承，并非只有一份。
<br/>　　但，参悟者，能获得何种层次的传承，全凭自身机缘以及天赋。
<br/>　　此时，在萧逸的感知中，每一个灵位上的传承，都极其强大。
<br/>　　最弱的，都在地极境以上。
<br/>　　猎妖殿作为最古老的势力，其底蕴果然强得发指。
<br/>　　萧逸没有浪费时间，盘膝坐下。
<br/>　　而后，再无动作，只是这般静坐着。
<br/>　　不同于其他传承，猎妖殿的传承，无法自我参悟。
<br/>　　而是自我放出感知，灵位上的传承，会自动寻来。
<br/>　　自身的感知，会与这里的传承，产生共鸣。
<br/>　　萧逸的感知，已然放出。
<br/>　　不过片刻，殿内的数千份传承，悉数涌来。
<br/>　　那一道道绿光，瞬间在萧逸周遭环绕，飞跃不停。
<br/>　　萧逸看得眼花缭乱，干脆闭上了眼睛，只凭感知，观察着这些绿光。
<br/>　　嘭…嘭…嘭…
<br/>　　一阵阵轰鸣声，陡然而起。
<br/>　　声音，并非在外头发生。
<br/>　　而是在萧逸的脑海中发生。
<br/>　　这些绿光，正在于萧逸的感知碰撞着，争锋着。
<br/>　　“嗯？”萧逸一惊。
<br/>　　每一道绿光上蕴含的力量，极其庞大。
<br/>　　数千道一起攻来，何其恐怖。
<br/>　　一阵头晕目眩的感觉，涌上萧逸心头。
<br/>　　渐渐的，这些晕眩，化作了痛苦。
<br/>　　这便是所谓的产生共鸣。
<br/>　　按照以往历代殿主的参悟情况，绿光会逐渐离去。
<br/>　　以减轻与参悟者感知的碰撞，以此来减缓参悟者的痛苦。
<br/>　　随着绿光逐渐离去，痛苦会逐渐减缓。
<br/>　　当然了，在这个过程中，率先离去的，会是蕴含力量最强的绿光。
<br/>　　随之递减，从强至弱，一个个离去。
<br/>　　待得绿光离去的数量，维持在参悟者能承受的范围时，参悟者方才开始正式参悟。
<br/>　　言归正传。
<br/>　　此时，萧逸的感知，正与这数千道绿光碰撞着。
<br/>　　正常来说，一般数千道绿光同时涌来，几乎是一瞬间，就会开始快速消减。
<br/>　　因为，没人能承受数千道绿光的同时‘攻击’。
<br/>　　只不过，萧逸却承受下来了。
<br/>　　感知，是武者的一种手段。
<br/>　　由武者的小世界赋予。
<br/>　　武者的几乎所有手段，都由小世界赋予，感知、御空飞行、操纵天地灵气等等。
<br/>　　也就是说，武者感知的强弱，与小世界强弱有关。
<br/>　　这便也是为何武者修为越高，感知越强的原因。
<br/>　　武者从踏入洞玄境开始，便有了气泉。
<br/>　　而气泉大小不变，小世界却会随着修为增高而变大，变强。
<br/>　　自然的，修为越高者，感知力越强。
<br/>　　当然了，决定感知强弱的，还有其它因素，小世界强弱，是比较重要的一点。
<br/>　　言归正传。
<br/>　　萧逸是个异数，庞大得恐怖的小世界之下，他的感知，强得发指。
<br/>　　这也是他以前修为低时，却总能越境界感知到别人修为的原因。
<br/>　　此时，数千道绿光齐齐攻击下，根本奈何不了他的强大感知，乃至心神。
<br/>　　不错，这些绿光，还有着侵蚀心神的特点。
<br/>　　这也是传承中的一项考核。
<br/>　　只不过，以萧逸剑心之坚如磐石，根本无法撼动。
<br/>　　此时，数千道绿光，正与他的感知争执着、僵持着。
<br/>　　那不断的剧烈碰撞，让得殿内灵气、禁制，悉数暴走。
<br/>　　传承殿之外。
<br/>　　殿主守候在此。
<br/>　　他看不到里头发生了什么，却听到了那一道道禁制暴走的轰鸣声。
<br/>　　“怎么回事？”殿主皱起了眉头。
<br/>　　“传承参悟，显然已经开始了。”
<br/>　　“可这禁制轰鸣，却久久持续。”
<br/>　　他当然也参悟过里头的传承。
<br/>　　那时的他，最终从3900道绿光中，留下了2800道绿光。
<br/>　　留下来的绿光，蕴含的传承，皆是传承殿内上等的传承。
<br/>　　那时的他，可是当时的猎妖殿第一天骄。
