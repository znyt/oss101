　　第302章
<br/>　　大风大雨过后，河面上到处漂浮着被风浪拍碎的船板，以及溺死在河中的尸体。
<br/>　　莫鸣米无可奈何的退兵，损兵折将无数。
<br/>　　还有一部分人，躲过了风浪，躲过了浅滩的毒蛇，最后被水冲到蛤蟆岛的岸边。
<br/>　　程大雷带着人赶到时，看到这些人脸色苍白，身上沾满烂泥，一见到蛤蟆寨的兵丁，就立刻举起双手。
<br/>　　“我们投降，我们投降啦！”
<br/>　　终于，蛤蟆寨有了第一批俘虏，接下来的问题，便是如何处理这批人。
<br/>　　程大雷命徐神机清点人数，把他们都聚到蛤蟆寨的演武场上。
<br/>　　数目很快统计出来，被冲到岸边的俘虏约有五百余人，现在这些人都像是霜打的茄子一般，不知等待自己的是怎样的结局。
<br/>　　“大当家，抓到一条大鱼。”徐神机兴奋的过来。
<br/>　　“什么大鱼？”
<br/>　　“审问清楚了，他叫严笛，是莫鸣米的军师。”
<br/>　　程大雷眼前一亮，挥挥手道：“快，快，把他带上来。”
<br/>　　严笛被绑着押上来，严笛其实也很倒霉，他和莫鸣米在同一艘船上，在风吹浪打的时候，一个不留神，从甲板上跌落水中，当时兵荒马乱的，谁也没有注意到他，他也就随着风浪被吹到岸边。
<br/>　　可虽然落在蛤蟆寨，严笛神情还是很倨傲，在程大雷面前立而不跪。
<br/>　　事实上，严笛能够很快被发现，也是因为身上这种出类拔萃的英气。
<br/>　　“呔，还不跪下！”高飞豹喝道。
<br/>　　严笛呵呵冷笑：“要杀便杀，何必废话，严某皱皱眉头，便不是七尺男儿。”
<br/>　　“好啦，好啦，两军对垒胜败自有天定，何必辱他。”程大雷挥挥手：“给严军师松绑。”
<br/>　　严笛松开绑绳，揉了揉发酸的手腕，目光看着周围。
<br/>　　倒是没有格外注意程大雷，关注点都在徐神机身上。只见他神情委顿，留着三撇胡须，看上去平平无奇。
<br/>　　唉，这才是真人不露相呐，输在这样一个人手中，却也不冤。
<br/>　　“严军师，想不想回去？”程大雷问。
<br/>　　“无非一死而已，何必拿这种话诓我，你们会放我回去，你们敢放我回去！”严笛打定输人不输阵的主意，在程大雷面前绝对不可坠了威风。
<br/>　　“我让你回去，是要你给莫将军递一封信，把我的话传过去。”
<br/>　　“你真会放我走？”世上有不怕死的，却没有不想活的，听到这话难免心头一动。
<br/>　　“当然啦，我杀你做什么。不过，要走也不是现在走，你还得将琴川关的情况详详细细，一五一十与我说清楚。”程大雷。
<br/>　　“休想，我严笛绝不会背叛莫将军。”
<br/>　　“当然啦，关于这点我也想到了。”程大雷一挥手：“放刘悲。”
<br/>　　严笛正恍神的光景，突然发现面前杵着一个人，在看到对方一张脸时，严笛恍然觉得，内心有某些东西被击破了。
<br/>　　生又何欢，死又何苦，王侯将相，不过是荒坟枯骨，自己劳劳于名利，究竟所为那般。
<br/>　　不过是一眼注视，严笛就像霜打的茄子般，精气神被抽走大半。
<br/>　　“刘悲，他就交给你了，什么时候把他肚子里的存货都搞明白了，再来见我。”
<br/>　　严笛被刘悲带人押下去了，接下来的日子里，他将接受刘悲无微不至的精神关怀。
<br/>　　接下来，程大雷开始处理山寨的俘虏。
<br/>　　对于这些人，自然是全部杀掉省事。但正所谓杀降不详，更何况，真若杀了他们，只会让自己落个残暴名声。
<br/>　　名声是个好东西，有时候也要好好经营。
<br/>　　程大雷带着徐神机等人走出聚义厅，来到俘虏聚集的校场。
<br/>　　只见人都东倒西歪的站着，表情麻木，眸子里闪烁着忐忑，恐惧，看到程大雷时，惧意更盛。
<br/>　　“诸位兄弟，诸位兄弟……”程大雷笑着摆摆手：“都坐，都坐，大家不必拘束。”
<br/>　　“坐！”
<br/>　　看守他们的汉子齐声大吼，众人吓得心惊胆战，只好席地坐下，眼睛都看着程大雷。
<br/>　　“诸位兄弟也别害怕，两军对垒，胜也好，败也好，和你们无关。当兵打仗，无非都是讨口饭吃而已。我和诸位往日无怨，近日无仇，不会杀诸位的。”程大雷和颜悦色的说着：“我知道各位都是附近的父老乡亲，你们可以打听打听，从我们兄弟到了后，可有欺负一家老弱，抢过一户穷人。可我们兄弟不招谁不惹谁，莫鸣米却要派大军来打我们。让弟兄们吃苦受罪，都是爹生父母养的，死了谁也是人命。”
<br/>　　“今日弟兄们落在这里，想留下入伙的，我蛤蟆寨好酒好肉，不欺负谁，也不受谁欺负。想走的也没关系，我们蛤蟆寨给路费，送你们回家。”
<br/>　　“您真肯放我们走？”有人忍不住问出声来。
<br/>　　程大雷笑而不语，拍了拍手，两口装满铜板的箱子抬了上来。
<br/>　　程大雷自己撤了，这里的事情交给徐神机处理，想留下的登记花名册，想走的发给路费。
<br/>　　“程大雷，你的法子没用。”
<br/>　　程大雷正在聚义厅歇着，李行哉跑了进来：“十个人能留下一两个就不错了，大多数人拿钱走了。”
<br/>　　“你懂什么。”程大雷抿口茶：“人走得越多越好，最好一个不剩下才好。”
<br/>　　“可你不仅花了钱，最好还什么都落不着，你图个什么呀？”
<br/>　　“图个名声呗。”程大雷慢条斯理的喝茶。
<br/>　　哪怕一个人都不留，程大雷这笔买卖都不亏。人散出去的越多，这件事就穿得越光，大家都会知道，被蛤蟆寨俘虏的人，不仅不会被杀，反而给回家的路费。那么，下次再有人来打蛤蟆寨，这些人还会拼命嘛，既然投降就可以活着，谁人还会拼命，当兵也不过为混口饭吃而已。
<br/>　　李行哉琢磨着程大雷的话，脸上露出若有所思的表情。
<br/>　　最后，五百俘虏，留下的不过一二百人，程大雷将他们分成四拨，编入山寨的四个小队中。至于秦蛮，因为还在落玉寨，暂且没有扩充兵员。
<br/>　　对于这些人的思想建设也不能放松，随着时间慢慢推移，想得到这帮人的忠诚也并不是什么难事。
<br/>
