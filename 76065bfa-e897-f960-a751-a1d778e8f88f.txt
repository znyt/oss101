　　第663章
<br/>　　张字军感受到了张肥的愤怒，更确切来说，是从张肥处感受到来自和珅的愤怒。
<br/>　　这支本来就暴躁的队伍变得更加如狼似虎，在凉州兵眼里，他们不是一个个人，更像是一头头十几天没见过肉味的饿虎。而自己在他们眼中，很显然成了美味可口的食物。
<br/>　　面对这样的状况，大家似乎除了撒丫子狂奔，便没有其他任何选择。他们畏惧张字军，不过张字军同样有畏惧的存在。
<br/>　　和珅下达的命令是活捉，还必须是全部，那么他们就不可能放走一个，不然和珅愤怒的后果估计也是他们不可承受的。
<br/>　　而此刻程大雷窝在屋子里，外面风声呼呼作响。今夜，其实也算是一场蛮紧张的战斗，毕竟若是农田被烧毁，对朔方城绝对是一场灾难。
<br/>　　程大雷其实也很担心，但他同时也需要学会放心。
<br/>　　漫漫长夜，如此熬过。
<br/>　　第二天天还没亮，和珅就来向程大雷报告。他很疲惫，但整个人也很激动。
<br/>　　“启禀大当家，幸不辱命，全都被捉住了，城里的探子和城外的凉州兵一共一百三十七人，过程中跑死两个，剩下的都在。”
<br/>　　程大雷正在洗脸，听到这消息后他也算松口气。
<br/>　　“没出事就好，还是要小心防范，他们这次没成功，说不得还要有下次。”程大雷顿了顿：“对了，抓住的人准备怎么处理？”
<br/>　　“当众处决，他们敢来朔方城找事，就必须付出代价。同时也是做给外人看，杀鸡儆猴。”
<br/>　　“啧，都杀了，一百多人呐。”程大雷顿了顿。
<br/>　　“大当家，不恨不行呐，今日不现雷霆手段，日后怕是会后患无穷。”
<br/>　　程大雷也情知和珅的话有道理，但说是如此说，真如果战场上杀敌破万，也没有什么。可这些人算是战俘，如果杀起来肆无忌惮，心里还是稍稍有些过不去。
<br/>　　但毕竟也是没办法的事。程大雷擦干净手脸，道：“你处理就好，处理得干净些。”
<br/>　　和珅已有和珅的想法，并且会坚持自己的想法。所以，程大雷就算不答应，他也不会轻易被说服，反而会想办法说服程大雷。
<br/>　　他已经开始着手处理这件事，那一百三十五名俘虏拉到城中心。在所有人面前，当众斩首。
<br/>　　朔方城聚集的，都是来自四方各地的亡命徒，但即便是亡命徒也被这一幕震慑住。那个面白无须，本来感觉毫无威胁的中年男人，却在此时让大家觉得怕。
<br/>　　原本还有些人不太安分，但在现在已开始忌惮，假若还想不安分，无端端生一些事出来，说不得下次被砍头的便是自己了。
<br/>　　在今日，程大雷并没有现身，但却也没有缺席。他出现在城主府的阁楼上，远远看着。
<br/>　　其实也看不太清楚，只是见和珅一声令下，便是上百颗人头滚滚而落。
<br/>　　青天白日，鲜血淋漓，便在这楼上，似乎都能嗅到些许的血腥味。
<br/>　　“这样……做有违天和呐。”徐神机叹了口气，说罢，他顿了顿道：“大当家，你知不知道他一步打算怎么做？”
<br/>　　“不知道，我没有问。”
<br/>　　“大当家怎么可以不问。”徐神机一惊，难以置信道：“你这放权有些太厉害了吧？”
<br/>　　程大雷当然明白徐神机话中的含义，但他无动于衷。他依旧注视着远方，道：“你想说什么吧？”
<br/>　　“嘿，我还真替大当家问了，他准备将这些人的人头送到凉州城，给宋伯康一些颜色看看。”徐神机叹道：“啧，杀鸡儆猴，原来儆的这只猴是宋伯康。”
<br/>　　程大雷当然不会不知道徐神机在做什么，他是要放权，但不是要当傻瓜。
<br/>　　“怕是要回蛤蟆城了。”
<br/>　　“怎样？”
<br/>　　“嗳，我们和宋伯康的好关系要结束了。”
<br/>　　……
<br/>　　凉州城，宋伯康坐在帅案后，脸黑得像锅底一般。
<br/>　　和珅果然说到做到，将一百余颗人头送到凉州城的城门前。宋伯康已经亲自看过，那当然不是一副悦人的场面。这明显是向自己示威，而且是不加掩饰的。
<br/>　　在大厅左右，是凉州城的文官武将，他们也见过了那些送回的人头。现在垂着头不说话，生怕那句话不对，便触犯宋伯康，令他将怒火发泄在自己身上。
<br/>　　这时候，宋伯康已经将自己的怒火压制下去，这当然不是一件容易的事情。
<br/>　　“诸位，对这件事有什么看法？”
<br/>　　很久没有人说话，半晌，陆洛大着胆子开口，毕竟，他身上的事也不差这一件两件。
<br/>　　“王上，这程大雷何其狂妄，已经留不得他了，要给他些颜色看看。”
<br/>　　“说得好。”宋伯康忍不住赞了一声，然后问：“怎么给他颜色看看？”
<br/>　　“呃……”陆洛一阵无语，憋了半天才道：“王上，是否要对程大雷动手？”
<br/>　　宋伯康无可奈何的看了他一眼，感觉自己的智商已被他拉低许多。
<br/>　　“一山不容二虎，一州之地，容不下两个说话做主的人。”宋伯康缓缓道：“我们和程大雷之间，已经到不得不到用刀枪说话的地步了。”
<br/>　　“是是，王上高见。那程大雷不知天高地厚，必然不能让他猖狂下去。”
<br/>　　众人连忙附和，尤其是陆洛，说话的声音最是响亮。
<br/>　　“怎么打？”宋伯康目光注视着他。
<br/>　　“这……”陆洛再一次无语。
<br/>　　宋伯康已经特别不想搭理他，昂起头，目光从大厅内每个人脸上扫过。
<br/>　　“我今日说的事，既关乎我自己，也关乎未来的前程。我们和程大雷已经到水火不相容的地步，诸位现在聊一聊，如何动手？”
<br/>　　诸人沉默着，然后有人开始发表自己的意见，接着开始议论纷纷。凉州城的文武官员，都是来自不同的家族，只有极少一部分，是宋伯康自己的亲信。
<br/>　　如果凉州丢了，各家的利益也会遭受损失。凉州在西北一地，有极重要的战略位置，大家都不能容忍失去凉州。
<br/>　　“王上，可不可以从京州借兵？”
<br/>　　半晌过后，戴白的声音盖过了所有人，宋伯康的目光落在他身上。
<br/>(本章完)
