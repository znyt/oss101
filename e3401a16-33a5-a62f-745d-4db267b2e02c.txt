　　这个位面肯定有能威胁到自己的存在，毕竟相对于喰种来说，尸兄的武力值，要稍高了一些。
<br/>　　但这才只是病毒刚刚出现的时候啊，这么早的时间，居然就有尸兄能让他产生危险感了？
<br/>　　甚至，他能感觉到，如果自己处理不好的话，很有可能在这阴沟里翻了船。
<br/>　　“嗡嗡嗡~”
<br/>　　令人头皮发麻的声波传来，像是有什么生物正在震动着翅膀，白小飞立在楼前，将长刀横在胸口。
<br/>　　不消几分钟的功夫，铺天盖地的蚊虫飞了出来，这些蚊子，都有成年人拳头大小，长相狰狞，头部硕大的口器，让人头皮微微一麻。
<br/>　　“冲过去，小心别被这些蚊子碰到，它们身上可能感染着病毒！”
<br/>　　白小飞怒吼一声，手中的长刀不断挥舞，将四周的蚊群斩落到地下。

<br/>　　像是想到了什么，刘哔击退了身边的蚊群，对大头尸兄发动了吞噬技能。
<br/>　　毫无反应，果然没死！
<br/>　　被这诡异的情况搞得有些束手无策，刘哔心中犹豫，还要不要继续进行下去。
<br/>　　虽然高风险代表的是高收入，但当风险已经超出自己的掌控范围之后，还要不要继续，就成了一个极其艰难的选择了。
<br/>　　可是，现在放弃的话，之前他所做的事情，可就全部白费了。
<br/>　　他只要一走，白小飞都不知道会不会从这所大厦中活下来，即使存活，小鹿和保安估计也没生存下来的希望了。
<br/>　　这对剧情影响，实在太大，他也不知道这样做，会带来什么改变。
<br/>　　最重要的，白小飞如果死亡，他都不知道要去什么地方找那些高阶尸兄。
<br/>　　毕竟他时间有限，说不定什么时候就要回归自己的位面，他总不能白来一趟吧。
<br/>　　咬了咬牙，刘哔心中下了决定，继续跟在白小飞身后。
<br/>　　他对自己的实力有信心，只要戒备一些，如果真会发生什么，情况不好的话，他逃走应该还是没有问题的。
<br/>　　……
<br/>　　“砰！”
<br/>　　一声清脆的响动，白小飞撞开走廊的大门，找到了电梯。
<br/>　　这段时间内，他们不知道斩落了多少变异蚊虫，大厅的地板上，几乎都铺满了厚厚的一层蚊子尸体。
<br/>　　但这种东西，就像是无穷无尽一般，不管他们怎么杀，数量都看不到有任何变化。
<br/>　　还是那样铺天盖地的，就像是永远杀不净一般。
<br/>　　“先进电梯！”
<br/>　　匆忙之中，白小飞摁下了电梯的按钮，幸运的是，电梯还能正常使用，好歹在这种关机时刻，没有掉链子。
<br/>　　背靠着电梯，众人将小鹿护在中间，艰难的抵御着身旁的变异蚊群。
<br/>　　“叮~”
<br/>　　几分钟的功夫，清脆的声音响起，电梯大门缓缓打开，众人瞬间窜了进去，按下了关闭按钮。
<br/>　　但因为蚊群拼死涌进，电梯门感应到物体，却是怎么都关闭不上。

<br/>　　“烈！”
<br/>　　一片火焰突然出现，呼啸着，将变异蚊群吞没，瞬间一股焦香味，和噼里啪啦的声音响起，门前瞬间被清理出了一片空缺。
<br/>　　借此机会，电梯门终于闭合，白小飞依靠在墙壁上，大口喘着粗气，一边向刘哔投来了感激的目光。
<br/>　　“小鹿，你家在几楼？”
<br/>　　手中的黑色烧火棍光芒减弱了下去，刘哔转过头来，看着已经被惊吓到说不出话来的小女孩。
<br/>　　即使是末世，她也是在众人的庇护之下一路走过来的，又哪里经历过这样恐怖的场景，现在还能站着，都已经很不可思议了。
<br/>　　“就……就从上面数第二行，右边的那个，小鹿也不知道，那是几楼。”
<br/>　　声音有些结巴，小鹿指着按钮的方向，刘哔看了一眼，摁下了十二楼的按钮。
<br/>　　随着一股轻微的失重感，电梯缓缓运转，众人紧悬着的心情，也微微开始放松下来。
<br/>　　除了刘哔。
<br/>　　这些蚊子对他来说不算什么，以及这个大厦中还有几只爬行者，那些东西也很好解决，
<br/>　　他在意的，只是那个给他带来危险感觉的尸兄。
<br/>　　不过看样子，对方并不打算动手，或者说并不打算在这里动手。
<br/>　　甚至连注没注意到他们，刘哔心中都不太清楚。

<br/>　　暗中嘀咕了一声，刘哔加深了心中的警惕，看着电梯表上代表楼层的红字慢慢跳动。

<br/>　　到达九楼的时候，电梯停止了运转，刚刚缓了一口气的白小飞和保安小哥，瞬间抄起了武器，看向头顶的方向。

<br/>　　指了指天花板，白小飞双手握刀，全神贯注的注视着上方。

<br/>　　令人牙酸的声音响起，随后是一股浓郁的腥臭味道，电梯的顶端，缓缓被腐蚀开了一个小洞，从洞口有藻绿色的液体滴落了下来。
<br/>　　“大家快让开，这液体有毒！”
<br/>　　白小飞心中一惊，不由自主的吼出声来，出声的同时，也极为麻利的躲向旁边，避免被绿色液体沾到。
<br/>　　就连电梯的钢板都阻挡不了的腐蚀性，他可不认为，自己的肉体，比这些钢板都要坚硬。
